= 列における垂直境界線の表示

次のコードは、Cell の Appearance の pick:[win-forms="link:{ApiPlatform}win{ApiVersion}~infragistics.win.appearance~bordercolor.html[BorderColor]"]  プロパティを使用して列境界線としての垂直線を表示する方法を示しています。

*Visual Basic の場合：*

----
Private Sub DisplayingVerticalBordersinColumns_Load(ByVal sender As System.Object, _
  ByVal e As System.EventArgs) Handles MyBase.Load
	Me.UltraGrid1.DisplayLayout.Override.CellAppearance.BorderColor = Color.Red
End Sub
----

*C# の場合：*

----
private void DisplayingVerticalBordersinColumns_Load(object sender, System.EventArgs e)
{
	this.ultraGrid1.DisplayLayout.Override.CellAppearance.BorderColor = Color.Red;
}
----
