= スタイルセットとリソースを使用

Application Styling Framework™ はリソースと呼ばれる独自のスタイリング ツールをサポートします。リソースは任意の状態に適用できる外観プロパティのコレクションです。Infragistics 製品をよく知っている場合には、リソースを外観オブジェクトとして認識できるでしょう。外観オブジェクトとリソースのこの関係は、生産性を向上させるために Presentation Layer Framework™ と Application Styling Framework が連動していることを示す重要な例です。

スタイル エクスプローラの [リソース] タブで新しいリソースを追加し、次に UI ロールの状態に適用することができます。UI ロールの状態をスタイルして、その状態からリソースを作成することもできます。この柔軟性によって最大限の効果が保証され、スタイルセットを作成するために必要な時間を削減します。
