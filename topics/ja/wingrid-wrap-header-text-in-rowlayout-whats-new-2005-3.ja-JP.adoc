= RowLayout におけるヘッダ テキストの折り返し

{ProductName} 2005 Volume 2 でこのプロパティを追加しました。ただし、通常モード（カード ビューではないモードおよび行レイアウトではないモード）に限って実装されました。Windows Forms 2005 Volume 3 では、行レイアウト通常ビュー モード、カード ビュー行レイアウトでないモード、およびカード ビュー行レイアウト モードのすべてのモードが実装されます。

WrapHeaderText が True に設定されている場合、ヘッダのキャプションに対応するために WinGrid はヘッダのキャプションを自動的に折り返し、ヘッダの高さを増します。これは通常、キャプションを完全に表示するために必要なサイズよりもヘッダの幅が小さい場合に発生します（たとえば、小さくサイズを変更した場合）。
