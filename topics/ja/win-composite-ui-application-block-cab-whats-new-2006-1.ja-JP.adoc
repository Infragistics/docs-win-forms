= Composite UI Application Block (CAB) 2006.1

Composite UI Application Block（CAB）フレームワークは、Microsoft Visual Studio® 2005（.NET Framework 2.0）をベースとしたソース コードベースのコンポーネントです。

次のリンクをクリックすると、Infragistics CAB Extensibility Kit™ の新しいアドオンについて説明したトピックを読むことができます。このアドオンは、CABフレームワークとのシームレスな統合を提供し、これにより豊富な機能を持つ {ProductName} ツールセットを使用して高度なスマート クライアント アプリケーションを開発することができます。
