= 新しい WinGanttView コントロール

今回のリリースでは、増加している Windows Forms コントロールのリストにインタラクティブな WinGanttView™ コントロールを追加しています。 WinGanttView コントロールは Microsoft® Project のコントロールと機能的に似ており、プロジェクト管理に対して同様のフロント エンドを提供します。

image::Images/WinGanttView_New_WinGanttView_Control.png[]

このコントロールによって、Tasks、Summary Tasks、Milestones、Deadlines、PercentComplete、Dependencies、Constraints および Tooltips に対するサポートを提供することでプロジェクト管理ソフトウェアを効果的に作成することができます。 グリッドの形の表形式のリストを提供するだけでなく、さまざまなプロジェクトの詳細をガント チャートで表します。 コントロールのチャート セクションの列ヘッダーは、WinTimelineView™ コントロールに似た時間の間隔と水平のスクロールバーを表示できます。 WinGanttView は WinSchedule コントロールによっても使用される WinCalendarInfo™ コンポーネントを通してその機能を取得します。
