= ProgressBar ツール

新しい ProgressBarTool クラスを使用すると、ツールバーまたはメニューにプログレス バーを表示できます。このツールは、進捗状況を示す部分の外観を制御するためのいくつかのプロパティを提供します。ProgressBar ツールには編集モードはありませんが、プログレス バーをクリックすると ToolClick イベントが、ダブルクリックすると ToolDoubleClick イベントがトリガーされます。
