= 新しいスパン タグと Style 属性

WinFormattedLinkLabel に追加された優れた機能のひとつが STYLE 属性です。この属性を使用すれば、前景色へのパディングから境界線スタイルまでスタイルを設定できます。STYLE 属性は、既存の <p> タグ、そして新しい <img>、<hr>、および <span> タグで使用できます。

<span> タグはインライン スタイルを表示するために特に作成されました。<span> タグはそれ自体では何も行いません。ただし、<span> タグにスタイル属性を追加し、多くのことを実行できます。背景色を変更、背景グラデーションを使用、境界線の色やスタイルを変更、<font> タグのようにフォント スタイルを編集、セクションの高さと幅を変更、余分なパディングとマージンを適用などを行うことができます。<span> タグで使用できるスタイルの完全なリストは、「Style 属性の使用」を参照してください。
