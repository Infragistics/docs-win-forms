= スポーク ペイン

[スポーク] ペインはアンカーの形状を編集するために使用されます。深さ値 100 とスイープ値 360 は円です。値が小さくなると、スポークはよりはっきりと表示されます。

pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedleanchor~spokedepth.html[深さ]"]  -- この値を 0 から 100 の整数に設定します。この値は、各スポークとラジアル ゲージの中心の間の距離です。

pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedleanchor~spokesweep.html[スイープ]"]  -- この値を 0 から 360 の整数に設定します。この値は各スポークの角度の測定です。アンカーの「スポーク」の外観は、スイープ値が 100 未満の時に限って表示されます。
