= Excel ファイルからの数式の値

Infragistics Excel Engine™ には、Excel® ワークシートに存在する数式から実際に計算した値を取得する機能があります。以前は、Excel 数式を構成する Excel セルのセル値にプログラムでアクセスを試みると、Excel 数式を表す実際のリテラル文字列を取得します。この新しい機能で、Excel Library のオブジェクト モデルで作成された数式または Microsoft® Excel から作成された既存の数式にアクセスするかどうかにかかわらず、数式の実際の評価された値を取得します。
