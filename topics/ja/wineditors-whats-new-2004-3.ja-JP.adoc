= WinEditors 2004.3

== エディタ ボタン

UltraWinEditors はエディタ ボタンを表示できるようになりました。エディタ ボタンが表示されるのは、テキスト領域の左右にあるエディタ エレメント内です。エディタ ボタンはイベントを生成し、このイベントを使ってカスタム編集インタフェースを呼び出したり、エディタ内に含まれる値を変更するなどの操作ができます。エディタ ボタンは 5 種類あります（標準ボタン、リピート ボタン、状態ボタン、スピン ボタン、ドロップダウン ボタン）。
