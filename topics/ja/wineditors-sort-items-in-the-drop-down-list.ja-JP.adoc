= ドロップダウンリストの項目をソート

このトピックは以下の WinEditor コントロールに適用されます。

* WinComboEditor
* WinFontNameEditor

デフォルトでは、WinEditor コントロールはドロップダウン リストの項目を並べ替えません。ただし、pick:[win-forms="link:{ApiPlatform}win.ultrawineditors{ApiVersion}~infragistics.win.ultrawineditors.ultracomboeditor~sortstyle.html[SortStyle]"]  プロパティを設定すれば、リストの項目を昇順または降順に並べ替えることができます。ドロップダウン リストの項目は、それが表示するテキストとは別の値を設定できるため、SortStyle プロパティは項目の値またはテキストのいずれかによる並べ替えをサポートします。
