= WinCalculatorDropDown について

WinCalculatorDropDown コントロールは、以下の機能を持つドロップダウンのカスタマイズ可能な計算機を提供します。

* 外観のサポート - アルファ ブレンディング、グラデーション、画像などを含め、外観が完全にサポートされます。
* 組み込みテーマ - 標準、Office 2000、Office XP、Office 2003、Visual Studio 2005、および Office 2007 のテーマをサポートします。
* インク対応 -  ユーザーは、FreeForm のインク入力と仮想キーボード入力を切り替て使用できます。
* オーバーフロー インジケータ - テキスト オーバーフロー インジケータのイメージはカスタマイズ可能で、テキスト全体を表示するツールヒントをサポートします。
* null テキスト - コントロールの現在の値が null のときにカスタム文字列を表示します。
* カスタマイズ可能なアクション - 計算機ボタン イベントとカスタム計算機ボタンを使用することで、ビジネスルールに合わせて開発者が指定した計算を実行できます。
