= ゲージの目盛について

ゲージのスケールの値を表示するために目盛をゲージに追加することができます。TickMarks プロパティには 2 つの属性があります。

* 副目盛
* 主目盛

両方を構成して特定の値に「目盛」を表示することができます。主目盛は副目盛よりも通常大きいです。これらの目盛はスケールで n ごとの値を強調表示するように構成します。
