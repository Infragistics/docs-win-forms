= 対数軸を持つチャートのゼロ データ点のサポート

算術的にゼロの対数は定義されていないので、ゼロ データ値が対数軸付きのグラフにデータを渡した場合、Infragistics の前バージョンは例外をスローしました。{ProductName} 2007 Volume 3 リリースでは、対数の軸のあるグラフで表示したいデータ セットの一部として供給されるゼロ データ ポイントを許可するために、0 の対数に別の値を設定することができます。

グラフが Log(0) のプロットを試みる場合、値を表す特定の値に AxisAppearance オブジェクトで新しい LogZero プロパティを設定することができます。ToolTip はそれでも "0" を表示します。

LogZero プロパティのデフォルト値は double.NaN です。このプロパティをデフォルト値にしておくと、例外がスローされます。
