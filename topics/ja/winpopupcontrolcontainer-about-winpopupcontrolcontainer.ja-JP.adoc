= WinPopupControlContainer について

WinPopupControlContainer エレメントは、ドロップダウン ウィンドウ内に任意の Windows Formsのコントロールを表示できる、Windows Formsのコンポーネントです。このエレメントは、ランタイムにユーザー インターフェイスを提供しませんが、DropDown メソッドが呼び出されたときに、指定されたコントロールを表示します。エレメントはドロップダウン ウィンドウを提供するために WinDropDownButton エレメントと使用されます。
