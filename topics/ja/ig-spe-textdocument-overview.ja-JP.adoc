= TextDocument の概要

=== はじめに

`TextDocument` クラスは、定義済みの文章校正に従ってテキストを解析するための Infragistics® Syntax Parsing Engine の単一テキスト ファイルを表します。

このトピックは、`TextDocument` クラスの概念的な情報とコード スニペットへのリンクを含みます。
