= WinTab でタブの方向を指定

このトピックは、WinTab™ コントロールのタブの方向だけでなく、Tab オブジェクトのテキストの方向を設定する方法を示します。これらの設定は WinTab コントロールのスマートタグから適用できます。また、pick:[win-forms="link:{ApiPlatform}win.ultrawintabcontrol{ApiVersion}~infragistics.win.ultrawintabcontrol.ultratabcontrolbase~taborientation.html[TabOrientation]"]  および pick:[win-forms="link:{ApiPlatform}win.ultrawintabcontrol{ApiVersion}~infragistics.win.ultrawintabcontrol.ultratabcontrolbase~textorientation.html[TextOrientation]"]  プロパティを使用してコードからも設定可能です。

*Visual Basic の場合：*

----
'WinTab ですべてのタブの方向を指定します 
Me.ultraTabControl1.TabOrientation = TabOrientation.BottomRight 
' タブでテキストの方向を指定します 
Me.ultraTabControl1.TextOrientation = TextOrientation.Horizontal
----

*C# の場合：*

----
//WinTab ですべてのタブの方向を指定します
this.ultraTabControl1.TabOrientation = TabOrientation.BottomRight;
// タブでテキストの方向を指定します 
this.ultraTabControl1.TextOrientation = TextOrientation.Horizontal;
----
