= アクセシビリティのサポートの改善

リハビリテーション法の第 508 条に基づいて、すべての政府アプリケーションは障害を持つ個人に適応するための規格に準拠する必要があります。Windows Forms コントロールのほとんどは、複数のリリースでアクセス可能です。

WinListView および WinWeekView は完全にアクセス可能なものとしてコントロールの残りに加わっています。これらのコントロールは AccessibilityObject を持ち、エンド ユーザーが必要とするアクセシビリティ サポートを得ることができます。
