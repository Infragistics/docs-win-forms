= スマート タグのサポート

Visual Studio 2005（.NET Framework 2.0）の新機能であるスマート タグ機能では、コントロールのデザイン画面を終了することなく、最も一般的に使われるプロパティやオプションにすばやくアクセスしたり、通常のタスクを実行したりできます。

生産性向上を図るため、Infragistics の Windows Formsコントロール/コンポーネントにもそれぞれ固有のスマート タグが装備されました。シングル クリックで、フォームからフォーカスを移動せずに、特定のコントロール/コンポーネントの最も一般的に使われるプロパティやオプションにアクセスできます。

その際、フォームからフォーカスを移動する必要はありません。次の図は WinChart のスマート タグの例です。
