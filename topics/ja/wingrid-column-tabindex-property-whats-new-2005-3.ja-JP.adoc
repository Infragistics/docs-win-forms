= 列の TabIndex プロパティ

デフォルトで、WinGrid はセルの位置に基づいてセルのタブ順序を決定します。UltraGridColumn の TabIndex プロパティを使用して、タブ キーが使用されている時に WinGrid が行のセルをトラバースする順序を制御することができます。プロパティのデフォルト値は *-1* です。これは列がデフォルトのタブ順序に従うことを意味します。これらの列のタブの順序がデフォルトのタブの順序に従う場合に、2 つ以上の列の TabIndex を同じ値に設定できます。一部の列だけに TabIndex を指定し、他の列では TabIndex をデフォルト値にしておくこともできます。このようなシナリオでは、TabIndex をデフォルト値以外に設定している列は、TabIndex を設定していない列よりも優先されます。
