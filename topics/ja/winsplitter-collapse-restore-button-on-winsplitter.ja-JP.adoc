= WinSplitter の縮小/復元ボタン

WinSplitter™ によってエンドユーザーは、フォームでコントロールをリサイズするためにスプリッター バーをクリックおよびドラッグできます。 WinSplitter で使用できる縮小/復元ボタンを使用して、縮小および復元することもできます。

pick:[win-forms="link:{ApiPlatform}win.misc{ApiVersion}~infragistics.win.misc.ultrasplitter~collapseuitype.html[CollapseUIType]"]  プロパティを None に設定するか、コードまたはプロパティ ウィンドウから WinSplitter コントロールのボタン ユーザー インターフェイスを非表示にすることができます。

*Visual Basic の場合：*

----
Me.ultraSplitter1.CollapseUIType = Infragistics.Win.Misc.CollapseUIType.None
----

*C# の場合：*

----
this.ultraSplitter1.CollapseUIType = Infragistics.Win.Misc.CollapseUIType.None;
----

image::images/WinSplitter_Collapse_Restore_Button_on_WinSplitter.png[]

WinSplitter コントロールは縮小できるので、WinSplitter の前にドックされたコントロールは 0 の範囲を持ち、ランタイムには表示されません。 これは、コントロールの Collapsed プロパティを使用することで達成できます。

*Visual Basic の場合：*

----
Me.ultraSplitter1.Collapsed = True
----

*C# の場合：*

----
this.ultraSplitter1.Collapsed = true;
----
