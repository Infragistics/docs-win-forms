= 値ペイン

[値] ペインによって、スケール上のレンジの開始点と終了点を指定できます。

pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.gaugerange~startvalue.html[開始]"]  -- この値を 0 から 360 の整数に設定します。この値は、スケール上のレンジの開始点を設定します。

pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.gaugerange~endvalue.html[終了]"]  -- この値を 0 から 360 の整数に設定します。この値は、スケール上のレンジの終了点を設定します。
