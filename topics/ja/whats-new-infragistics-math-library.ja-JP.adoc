= Infragistics Math ライブラリ

{ProductName} の 2011 Volume 2 以降、Infragistics Math Library は高速、正確そしてメモリ効率のいい広範な数学演算を実行することができる中核的な関数の統一されたセットを提供します。さらに、このライブラリは以下を実行できます。

* データ構造を複数形式に変換せずに数学演算を実行するアプリケーションを作成できます。
* 数値メソッドを使用
* コンパートメント モデルをビルド
* モンテカルロ シミュレーションを実行
* 回帰問題を解決

詳細については、 link:infragistics-math-infragistics-math-library.html[「Infragistics Math Library」]を参照してください。
