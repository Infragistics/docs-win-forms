= テクスチャ ブラシ エレメント

ゲージ上のプロパティの背景を、キャンバスなどのテクスチャされたルック アンド フィールにしたい場合には、ブラシ タイプを [テクスチャ] に設定します。テクスチャ ブラシ エレメントを選択すると、開始色、終了色、テクスチャ スタイルを指定できます。以下を使用して、ゲージのプロパティにテクスチャ ブラシ エレメントを適用できます。

* link:wingauge-apply-brush-elements-using-the-gauge-designer.html[ゲージ デザイナ]
* link:wingauge-apply-brush-elements-at-design-time.html[ゲージ コレクション エディタ]
* link:wingauge-apply-brush-elements-at-run-time.html[コード]

以下のスクリーンショットは、以下の設定で目盛盤の BrushElement プロパティを TextureBrushElement に設定した状態のラジアル ゲージを表示します。

* テクスチャ スタイル -- キャンバス
* 開始色 -- シルバー
* 終了色 -- Dim Gray
