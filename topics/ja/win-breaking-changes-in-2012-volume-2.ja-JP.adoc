= 2012 Volume 2 の重大な変更

== .NET CLR 4 のみのサポート

=== 重大な変更

2012 Volume 2 リリースより, すべての {ProductName} コントロールとコンポーネントは .NET Framework (CLR) バージョン 4 のみでコンパイルされます。 すべての新規およびアップグレード済みの Infragistics 2012.2 は CLR4 をターゲットにする必要があります。 Visual Studio 2010 および 2012 の両方がサポートされます。
