= 数学関数

Infragistics Math Library™ は、さまざまな数学的オブジェクトに使用できる数学的関数の汎用クラスを提供します。

== クラスとメソッド

link:{ApiPlatform}math{ApiVersion}~infragistics.math.compute.html[Compute] は静的クラスで、 link:http://msdn.microsoft.com/en-us/library/system.math.aspx[System.Math] クラスの関数の拡張として機能するメソッドを含みます。

link:{ApiPlatform}math{ApiVersion}~infragistics.math.compute.html[Compute] クラスのメソッドには、Infragistics Math Library の異なるデータ タイプ (複素数、ベクトル、行列) に対するオーバーロードがあります。
