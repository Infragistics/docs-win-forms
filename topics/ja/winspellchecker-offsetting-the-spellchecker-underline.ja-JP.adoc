= SpellChecker の下線のオフセット

== トピックの概要

=== 目的

このトピックでは、スペルミスの単語の下に表示される下線のオフセットを変更する方法を紹介します。

=== SpellChecker の下線の構成

link:{ApiPlatform}win.ultrawinspellchecker{ApiVersion}~infragistics.win.ultrawinspellchecker.ultraspellchecker~underlineverticaloffset.html[UnderlineVerticalOffset] プロパティをスペルミスの単語の下線のオフセットを垂直方向に設定するための整数値 (正の値または負の値) に設定できます。

以下の例では、プロパティを設定すると SpellChecker の下線が変更されます。この機能をデモするために「underlyne」の単語のスペルを間違えます。下線は下に移動されました。

[options="header", cols="a,a"]
|====
|_UnderlineVerticalOffset=0_   (default)| _UnderlineVerticalOffset=5_  

|image::images/Offsetting_the_SpellChecker_Underline_1.png[]
|image::images/Offsetting_the_SpellChecker_Underline_2.png[]

|====

=== SpellChecker の下線の設定

プロパティ グリッドで `UnderlineVerticalOffset` を構成します。

image::images/Offsetting_the_SpellChecker_Underline_3.png[]

コードで `UnderlineVerticalOffset` を構成します。

*C# の場合:*

[source,csharp]
----
ultraSpellChecker1.UnderlineVerticalOffset = 5;
----

*Visual Basic の場合:*

[source,vb]
----
ultraSpellChecker1.UnderlineVerticalOffset = 5
----
