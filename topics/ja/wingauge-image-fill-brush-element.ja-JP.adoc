= 画像塗りつぶしブラシ エレメント

ゲージ上のプロパティの背景を画像として表示したい場合には、ブラシ エレメントを [画像の塗りつぶし] に設定します。[画像の塗りつぶし] エレメントを選択すると、選択の画像を指定できます。以下を使用して、[画像の塗りつぶし] エレメントをプロパティに適用することができます。

* link:wingauge-apply-brush-elements-using-the-gauge-designer.html[ゲージ デザイナ]
* link:wingauge-apply-brush-elements-at-design-time.html[ゲージ コレクション エディタ]
* link:wingauge-apply-brush-elements-at-run-time.html[コード]
