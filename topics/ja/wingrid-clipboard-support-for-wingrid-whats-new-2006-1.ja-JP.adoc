= WinGrid のクリップボード サポート

ワープロ アプリケーションやテキスト編集アプリケーションで [コピー] と [貼り付け] の機能が非常によく使用されることを考慮して、WinGrid にクリップボード機能を追加しました。WinGrid 自体でのクリップボードのサポートに加えて、WinGrid は Microsoft® Excel® のクリップボードをサポートできるようになりました。したがって、項目を WinGrid へ、または WinGrid からコピー、貼り付け、切り取りできるようになりました。同じように、WinGrid では Delete、Undo、Redo もサポートしています。デフォルト キー アクションでは標準のショートカット キー（例えば、Ctrl+C で Copy、Ctrl+V で Pasteなど）と同じように設定されています。WinGrid コントロールを使用すると、個々のセル、または複数のセルを WinGrid から Excel へ、またはタブ区切りセル値をサポートしているその他の表計算ソフトにコピーすることができます。WinGrid の Override クラスの AllowMultiCellOperations プロパティを"All"に設定するだけで、エンド ユーザーはデフォルトのキー操作マッピングを通じてコピーと貼り付けを行えます。
