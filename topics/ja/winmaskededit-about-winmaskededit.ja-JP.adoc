= WinMaskedEdit について

WinMaskedEdit は、Windows Formsのコントロールのひとつで、Windows Formsで強力な、カスタマイズ可能な入力マスクを提供します。

* 外観のサポート - アルファ ブレンディング、グラデーション、画像などを含め、完全な外観がサポートされます。
* 入力マスク -- 入力マスクをカスタマイズして使用すると、エンド ユーザーがフォームに入力するデータに割り当てられるフォーマットを制御できます。
