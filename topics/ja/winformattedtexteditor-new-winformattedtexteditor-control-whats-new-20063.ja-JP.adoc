= 新しい WinFormattedTextEditor コントロール

{ProductName} 2006 Volume 3 では、WinFormattedLinkLabel の編集コントロールである WinFormattedTextEditor コントロールが追加されました。WinFormattedTextEditor コントロールを使用すれば、WinFormattedLinkLabel が表示できるフォーマットされたテキストを表示できますが、編集のサポートが追加されています。

WinFormattedTextEditor コントロールには、Infragistic エディタの期待された基本的な編集機能と、WinFormattedTextEditor で認められる複数の高度な機能が含まれます。標準の KeyActionMappings を使用して、ナビゲーション、コンテンツの修正（クリップボードのサポートを含む）、フォーマッティングなどの作業を実行します。

切り取り、コピー、貼り付け、削除、および他のコンテンツ修正などの機能をサポートできる標準コンテキスト メニューがデフォルトで実装されています。ContextMenuItems プロパティを使用してメニューに表示したいことを決定できます。
