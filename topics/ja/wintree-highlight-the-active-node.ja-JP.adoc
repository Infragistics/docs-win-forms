= アクティブなノードを強調表示する

このトピックでは、ツリーのアクティブ ノードを強調表示する方法と、ツリーにおけるノードの位置に応じてその外観を変える方法について説明します。この例では、アクティブなノードが赤い背景色で表示されます。レベル 1 の子ノード（ルート ノードの子）がアクティブな場合、赤い背景色に加え、テキストが太字で表示されます。最後に、このコードではキーが "Node3" となっているノードも強調表示します。このノードがアクティブになると、上記の強調表示に加え、前景色が黄色で表示されます。
