= 新しい WinPanel コントロール

WinPanel™ コントロールは Infragistics® Windows Forms コントロールのリストへの追加機能です。WinPanel コントロールは、スクロールや自動サイズなどの .Net Panel コントロールの機能だけでなく、Presentation Layer Framework™ によって導入されたスタイリングの柔軟性を備えています。今まで、InboxControlStyler™ を .Net Panel コントロールと併用することは制限されており、それによりパネルの外観を制御する能力が制限されました。

WinPanel は UltraControl から派生するため、スタイリングに基づいた pick:[win-forms="link:{ApiPlatform}win{ApiVersion}~infragistics.win.appearance.html[Appearance]"]  を背景だけでなくスクロールバーにも適用できます。
