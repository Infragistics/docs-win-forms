= すべての立体チャートは新しいレンダリング エンジンを使用

2006 Volume 1 リリースでは、 link:winchart-new-3d-chart-rendering-engine-whats-new-2006-1.html[新しい立体チャート レンダリング エンジン] を活用するように既存の立体チャートを再デザインするだけでなく、多数の新しい立体チャートを提供しました。

本リリースでは、既存の立体チャートの再デザインが完了したので、すべての立体チャートは改善された立体レンダリング エンジンの能力を活用できるようになりました。今回大幅に改善された残りの立体チャートの画像サンプルを表示するには以下の各リンクをクリックしてください。

* link:images/winchart-all-3d-charts-use-the-new-rendering-engine-whats-new-2006-2-03.png[エリア]
* link:images/winchart-all-3d-charts-use-the-new-rendering-engine-whats-new-2006-2-04.png[ヒート マップ]
* link:images/winchart-all-3d-charts-use-the-new-rendering-engine-whats-new-2006-2-05.png[折れ線]

レンダリング品質の劇的な向上の例は、以下の立体折れ線グラフのスクリーンショットで示されています。左側はオリジナルの立体エンジンを使用してレンダリングされた折れ線グラフの例で、右側は新しい立体レンダリング エンジンを使用してレンダリングされた同じ折れ線グラフの例です。新しい立体チャート レンダリング エンジンでレンダリングされた、立体折れ線グラフのレンダリング品質と外観を見ると、陰と光の効果が大きく改善されていることを含み、大幅に改善されたことが分かります。
