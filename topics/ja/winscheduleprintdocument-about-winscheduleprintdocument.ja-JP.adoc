= WinSchedulePrintDocument について

WinSchedulePrintDocument は、WinSchedule ビューを簡単に印刷するための Windows Forms コンポーネントです。

* 印刷プレビュー -- この WinSchedulePrintDocument コンポーネントを WinPrintPreviewDialog コンポーネントとともに使用して、印刷前に WinSchedule ドキュメントのプレビューを確認できます。詳しくは、 link:winprintpreviewdialog-winscheduleprintdocument-using-winprintpreviewdialog-with-winscheduleprintdocument.html[「WinSchedulePrintDocument で WinPrintPreviewDialog を使用」]を参照してください。
* ヘッダ/フッタ -- WinSchedule ビューとともにカスタムのヘッダとフッタを印刷するように指定できます。
