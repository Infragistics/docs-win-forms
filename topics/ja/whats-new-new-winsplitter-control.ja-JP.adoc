= 新しい WinSplitter コントロール

WinSplitter™ コントロールを使用してランタイムにドッキングされたコントロールをリサイズできます。 提示するデータの長さが多様なコントロールがあるフォームに最適です。 このコントロールの優れた機能のひとつは、コントロールで使用可能な縮小/復元ボタンを使用して、分離したコントロールを縮小および復元する機能です。 Appearance オブジェクトおよび Appstyling を使用して、その他の任意の {ProductName} コントロール スイートと同じように WinSplitter コントロールをスタイルできます。
