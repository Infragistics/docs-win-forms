= WinExplorerBar でグループと項目を無効にする

このトピックは、UltraWinExplorerBar™ でグループと項目を無効にする方法を示します。

*Visual Basic の場合：*

----
' グループを無効にします 
ultraExplorerBar1.Groups(0).Enabled = False
' 項目を無効にします 
ultraExplorerBar1.Groups(1).Items(0).Settings.Enabled = Infragistics.Win.DefaultableBoolean.[False]
----

*C# の場合：*

----
// グループを無効にします
ultraExplorerBar1.Groups[0].Enabled = false;
 // 項目を無効にします
ultraExplorerBar1.Groups[1].Items[0].Settings.Enabled = Infragistics.Win.DefaultableBoolean.False;
----
