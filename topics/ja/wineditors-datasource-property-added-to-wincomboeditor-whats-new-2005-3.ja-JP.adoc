= WinComboEditor に追加された DataSource プロパティ

新しい DataSource プロパティを通じて、WinComboEditor コントロールをデータソースをバインドできるようになりました。データ バインディングを可能にするためには、WinComboEditor の DataSource プロパティを設定し、次のメンバを指定する必要があります。

* DataMember
* DisplayMember
* ValueMember

これで、WinComboEditor はデータ ソースの値を表示して、DisplayMember プロパティを通じて公開されます。 バインドされた ValueListItem を作成するとき、項目ごとに 1 回ずつ InitializeDataItem イベントが発生します。各イベントを処理することで、値に基づく Appearance 設定を個々の項目に適用できます。
