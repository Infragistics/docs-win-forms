= VBA コードで Excel ワークブックをサポート（XLSM フォーマット）

Infragistics Excel Engine™ は、VBA コードが組み込まれた Microsoft® Excel® ワークブックのロードが可能となりました。以前は、VBA コードのあるワークブックを保存すると、すべての VBA コードが失われました。新しい機能では、再び保存する時にロードされたワークブックから VBA コードをラウンドトリップすることができます。
