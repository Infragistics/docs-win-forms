= 垂直スクロール

*ScrollToLastItem* -- 垂直スクロールの動作は前バージョンのコントロールと同じになります。つまり、最後の表示ノードが一番上のノードになるまでコントロールをスクロールできます。

*ScrollToFill* -- 最後の表示ノードが完全に表示された位置より上にスクロールされないように垂直スクロールの範囲が制限されます。
