= CellClickAction プロパティの追加

このリリース以前は、 CellClickAction プロパティが存在するのは UltraGridOverride クラスのみでした。今回、CellClickAction が UltraGridColumn クラスに追加されました。このプロパティによって、ユーザーがセルをクリックしたときに発生する動作（列ごとの）を制御できます。たとえば、特定の列にあるセルをユーザーがクリックすると、セルが選択され強調表示されるように指定できます。
