= 追加された新しい Treemap グラフ

新しいツリーマップ グラフを WinChart™ コンポーネントに追加しました。

ツリーマップ グラフは、四角形の領域に大きい階層のデータ セットを表示するために使用されます。ツリーマップ グラフは多くのディレクトリ レベルに数千のファイルが格納されているハードドライブのコンテンツを視覚化するために通常使用されます。
