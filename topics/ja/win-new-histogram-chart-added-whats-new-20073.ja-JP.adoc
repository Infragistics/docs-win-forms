= 追加された新しいヒストグラム チャート

{ProductName} 2007 Volume 3 リリースから開始しましたが、新しいヒストグラム チャートが WinChart™ コンポーネントに追加されました。

ヒストグラム チャートは結果がどういう割合で各カテゴリーに分類されるのかを示す表のグラフ表示です。ヒストグラム チャートはデータの通常の分布曲線も表示できます。
