= 立体ポイント チャート

立体ポイント チャートは既存の link:chart-scatter-chart.html[散布図]に基づいており、全く別個のデータ シリーズの数値間の関係を本質的に示すので、科学的分析および予測解析のための有用な情報を提供します。立体ポイント チャートは、チャートにもうひとつの次元（z 軸）を追加することで散布図よりもさらに一歩先んじています。これによって、チャートを表示するアイコンに目をひくような深さが与えられます。
