= ツールの StatusText

ツールをアクティブ化すると、UltraToolbarsManager と同じコンテナにある UltraStatusBar の AutoStatusText タイプのパネルに、テキストが自動的に表示されます。以前のバージョンでは、この値はアクティブ化されたツールの解決されたツールチップに基づいて設定されました。個々のツールの SharedProps プロパティの StatusText プロパティをパネルに表示されるテキストに設定すると、ツールチップとステータス テキストに表示されるテキストを変更できます。
