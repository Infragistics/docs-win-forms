= ツールの間へのセパレータの追加

セパレータは、ツールの機能別グループを視覚的に示すために使用されるユニークなツールです。セパレータをデザインタイムに追加するには、WinToolbarsManager を右クリックして [ユーザー設定] を選択します。ツールバーをすでに作成していてそこにツールを追加していれば、前にセパレータを配置したいツールを右クリックして、コンテキストメニューから [グループの始まり] を選択します。これで、ツールの前にセパレータが配置されます。[ユーザー設定] ダイアログのツール リストで表示される場所ではなく、ツールバー自体でツールを右クリックする必要があります。
