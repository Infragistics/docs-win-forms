= 効果の相対バウンド

[相対バウンド] ペインは、ゲージ上の Needle ブラシ要素のサイズと位置を決定します。

X -- この値は Gauge コントロールの左側からの針ブラシ要素の距離を決定します。

Y -- この値は Gauge コントロールの上側からの針ブラシ要素の距離を決定します。

Width -- この値は針ブラシ要素の幅を決定します。

Height -- この値は針ブラシ要素の高さを決定します。

Units -- この値は針ブラシ要素の測定の単位を指定します。針ブラシ要素は、ピクセルまたは Gauge コントロールのパーセントで測定できます。
