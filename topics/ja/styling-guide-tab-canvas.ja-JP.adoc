= Tab キャンバス

Tab キャンバスに WinTab コントロールを伴うすべてのスタイリング修正を表示します。 このキャンバスには、ユーザー固有のアプリケーションで遭遇する可能性がある、複数の一般的な構成で Tab コントロールが表示されます。 Tab キャンバスに以下のコントロールがあります。

* WinTabStripControl
* WinTabControl
