= 新しいアセンブリ従属性

アプリケーション スタイリングは素晴らしい概念で、多くの開発者/グラフィック デザイナは利益を得ています。ただし、これらの利益を得るには、コントロールの一部を改善する必要があります。これらの変更のひとつには、 WinToolbarsManager および WinSchedule コントロールおよびコンポーネントにおける特有のダイアログ ボックスが関係します。これらのダイアログ ボックスの変更においては、他のいくつかの Infragistics アセンブリを参照しました。したがって、これらのコントロールを配備する場合、以前は必要ではなかった他のアセンブリを配備する必要があります。以下は、コントロールのアセンブリと共にこれらの 2 つのコントロールを配備する必要があるアセンブリのリストです。

== Infragistics.Win.UltraWinToolbarsManager

* Infragistics.Win.Misc
* Infragistics.Win.UltraWinEditors
* Infragistics.Win.UltraWinStatusBar
* Infragistics.Win.UltraWinTabControl

== Infragistics.Win.UltraWinSchedule

* Infragistics.Win.Misc
* Infragistics.Win.UltraWinEditors
* Infragistics.Win.UltraWinTabControl
* Infragistics.Win.UltraWinToolbars
