= WinPictureBox について

WinPictureBox コントロールは、.NET Framework で提供される組み込みの PictureBox と同様にひとつのイメージを表示するよう設計されていますが、以下のような追加機能があります。

* 外観のサポート - アルファ ブレンディング、グラデーション、画像などを含め、外観が完全にサポートされます。
* バインド可能な画像 - Image プロパティをバインドできます。Image プロパティが設定されると、ImageChanged イベントが起動されます。
* 画像の拡大/縮小 - 画像の拡大/縮小をサポートします。
* ドロップシャドー - 画像の下にドロップシャドーを表示できます。
