= WinGridColumnChooser について

WinGridColumnChooser コントロールは、エンド ユーザーが、Microsoft® Outlook® のフィールドの選択の動作と同様の方法で、列の表示と非表示を切り替えられるグリッド列のリストを提供します。エンド ユーザーは、各列の前にあるチェックボックスの選択または選択解除を行うことで、列の表示と非表示を切り替えられます。さらに、エンド ユーザーは、列を列の選択からグリッドに、またはその逆のドラッグを行うことにより、列を配置できます。
