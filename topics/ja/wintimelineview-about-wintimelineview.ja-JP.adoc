= WinTimelineView について

WinTimelineView™ コントロールは、時間スケジュールのための完全なソリューションです。 水平のタイムラインに沿ってタイムスロットおよび予定が表示されます。 すべての既存の WinSchedule コントロールと同様、WinCalendarInfo および WinCalendarLook コンポーネントに同様のサポートを提供します。

すべての WinSchedule™ コントロールに存在する pick:[win-forms="link:{ApiPlatform}win.ultrawinschedule{ApiVersion}~infragistics.win.ultrawinschedule.appointment.html[Appointments]"] 、pick:[win-forms="link:{ApiPlatform}win.ultrawinschedule{ApiVersion}~infragistics.win.ultrawinschedule.owner.html[Owners]"] 、および pick:[win-forms="link:{ApiPlatform}win.ultrawinschedule{ApiVersion}~infragistics.win.ultrawinschedule.timeslot.html[TimeSlots]"]  などの主要な機能に加えて、WinTimelineView コントロールによって時間を分または時間にスパンしたり、日付を日、週、月、または年にスパンすることができます。

WinTimelineView のグラフィカル表示は WinDayView™ に似ていますが、Owners は列ではなく行として配置され、TimeSlots は行ではなく列として配置されます。 その結果、予定の期間は WinDayView に表示される高さと反対に、WinTimelineView の幅に渡って表示されます。

WinTimelineView の機能の一部としては以下が挙げられます:

* 休日、休日の選択、繰り返しの予定、編集、リサイズ、および予定のドラッグ アンド ドロップのサポートを提供します。
* エンドユーザーは Owners の表示/非表示を変更するために行をリサイズでき、それによって表示できるオーナーの数を増減します。
* エンドユーザーは TimeSlots の表示/非表示を変更するために列をリサイズすることもでき、それによって表示できるタイムスロットの数を増減します。
* WinTimelineView コントロールの表示可能な領域に異なる Owners および TimeSlots を表示するために垂直および水平のスクロールバーを使用できます。
* 予定のドラッグおよびリサイズも可能です。
* WinTimelineView はキーボード ナビゲーションの完全なセットをサポートします。この場合、エンドユーザーはシンプルなキーボード ストロークを介して多くのタスクを実行できます。 アプリケーションがキーボード ナビゲーションを許可するとエンドユーザーのデータ入力および選択は必ず大幅に改善されます。
* 組み込み可能なエディターをサポートします。
* Application Styling™ のフルサポートが提供されます。
