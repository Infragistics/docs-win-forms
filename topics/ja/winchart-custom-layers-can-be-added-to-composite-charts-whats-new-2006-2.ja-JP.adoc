= カスタム レイヤを合成チャートに追加可能

2006 Volume 2 リリースは合成チャートにレイヤリング アーキテクチャを公開します（このアーキテクチャでは各レイヤが互いの上にプレスされて最終チャートが完成）。標準（非合成チャート）に加えて、合成チャート間でカスタム チャート レイヤを追加および削除するためのパワーと柔軟性をテキストにすることができます。
