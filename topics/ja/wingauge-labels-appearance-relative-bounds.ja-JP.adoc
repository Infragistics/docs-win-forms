= ラベル外観の相対バウンド

[相対バウンド] ペインは、ゲージ上の Labels ブラシ要素のサイズと位置を決定します。

X -- この値は Gauge コントロールの左側からのラベル ブラシ要素の距離を決定します。

Y -- この値は Gauge コントロールの上側からのラベル ブラシ要素の距離を決定します。

Width -- この値はラベル Brush 要素の幅を決定します。

Height -- この値はラベル Brush 要素の高さを決定します。

Units -- この値はラベル Brush 要素の測定の単位を指定します。ラベル Brush 要素は、ピクセルまたは Gauge コントロールのパーセントで測定できます。
