= WinPrintDocument について

WinPrintDocument は Windows Formsのコンポーネントのひとつで、WinPrintPreviewDialog とともに使用され、WinGridPrintDocument コンポーネントおよび WinSchedulePrintDocument コンポーネントのベースとなります。
