= ImageList キー

Visual Studio 2005 (.NET Framework 2.0) は、ImageList で項目に文字列名を指定できます。Appearance オブジェクトの Image プロパティが変更され、関連する ImageList の画像のキーを設定できるようになりました。これまで Image プロパティには画像そのものか、ImageList の Images コレクション内の画像のインデックスを示す整数のどちらかしか設定できませんでした。
