= DataGridView での埋め込みエディタの使用

Visual Studio 2005（.NET Framework 2.0）の DataGridView コントロールではカスタムの派生列を作成できます。Infragisticsでは、Infragistics の埋め込みエディタをホストする派生列が用意されています。エディタのタイプごとに個別の列クラスがあり、それぞれが（EmbeddableEditorBase から派生した）既存の埋め込みエディタ クラスに対応します。列はエディタ コントロールごとではなくエディタごとに存在します。たとえば、CurrencyEditor と NumericEditor はどちらも EditorWithMask を使用するので、これらのコントロール用の列タイプはありません。派生列はランタイムにもデザイン時にもグリッドに追加でき、バインドされていてもされていなくてもかまいません。
