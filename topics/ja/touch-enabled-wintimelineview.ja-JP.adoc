== トピックの概要

=== 目的

このトピックは、 _Infragistics_   のタッチ対応  _WinTimelineView_™ コントロール要素を紹介します。

== _WinTimelineView_   タッチ対応要素とジェスチャ

=== タッチ対応要素

以下のスクリーンショットは、 _WinTimelineView_   コントロールのタッチ対応要素を示します。タッチ要素は、タッチ対応モードで若干大きく表示されます。

image::images/Touch_WinTimelineView_1.png[]

=== ジェスチャ サポート

_WinTimelineView_   コントロールのジェスチャ サポートには、垂直および水平パン、フリック、タイムスロット記述子のドラッグ選択、予定のドラッグ操作が含まれます ( link:touch-gestures.html[タッチ ジェスチャ]を参照)。
