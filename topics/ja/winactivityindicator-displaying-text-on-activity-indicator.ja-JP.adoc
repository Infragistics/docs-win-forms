== _WinActivityIndicator_   にテキストの表示

[[_Ref341518673]]

=== 概要とプレビュー

13.1 以前のバージョンでは、 _WinActivityIndicator™_   は、アニメーション再生時のテキスト表示のサポートはなく、アニメーションのみのサポートでした。以下の画像では、アニメーション再生時に  _処理しています…_  テキストを表示し、アニメーションを停止時には「停止」テキストを表示するアクティビティ インジケーターを示します。

アニメーションを再生しています

image::images/WinActivityIndicator_Displaying_Text_on_Activity_Indicator_1.png[]

アニメーションを停止しました

image::images/WinActivityIndicator_Displaying_Text_on_Activity_Indicator_2.png[]

.注:
[NOTE]
====
コントロールの `UseOsThemes` プロパティはデフォルトで Off に設定されます。On に設定する場合、アニメーション テキストをオーバーレイします。アプリケーションをスタイルするには、Infragistics link:styling-guide-application-styling-framework-asf.html[AppStylist]® を使用することができます。
====

[[_Ref341518679]]

=== プロパティ設定

_WinActivityIndicator_   コントロールは link:https://docs.microsoft.com/en-us/dotnet/api/system.windows.forms.control.text[Text] および link:{ApiPlatform}win{ApiVersion}~infragistics.win.ultraactivityindicator.ultraactivityindicator~animationenabledtext.html[AnimationEnabledText] の 2 つのプロパティを公開します。`Text` プロパティは、アニメーションを再生していないときにテキストを表示します。`AnimationEnabledText` プロパティは、アニメーションを再生しているときにテキストを表示します。

以下のシナリオはテキストの解決処理を説明します。

1.`AnimationEnabled = True` の場合

* コントロールが null 値または空ではない場合、`AnimationEnabledText` プロパティの値を表示します。

* コントロールは、`AnimationEnabledText` プロパティが null 値または空の場合、`Text` プロパティの値を表示します。

2.`AnimationEnabled = False` の場合

* コントロールは `Text` プロパティの値を表示します。

コードまたは Visual Studio デザイン ビューにプロパティを設定できます。

*C# の場合:*

[source,csharp]
----
ultraActivityIndicator1.Text = "Stopped";
ultraActivityIndicator1.AnimationEnabledText = "Please wait while processing...";
----

*Visual Basic の場合:*

[source,vb]
----
ultraActivityIndicator1.Text = "Stopped"
ultraActivityIndicator1.AnimationEnabledText = "Please wait while processing..."
----
