= WinTree でノードを無効にする

このトピックは、WinTree™ でノードを無効にできる方法を示します。pick:[win-forms="link:{ApiPlatform}win.ultrawintree{ApiVersion}~infragistics.win.ultrawintree.ultratreenode.html[UltraTreeNode]"]  オブジェクトの pick:[win-forms="link:{ApiPlatform}win.ultrawintree{ApiVersion}~infragistics.win.ultrawintree.ultratreenode~enabled.html[Enabled]"]  プロパティを False に設定する必要があります。

*Visual Basic の場合：*

----
Me.ultraTree1.Nodes(0).Nodes(1).Enabled = False
----

*C# の場合：*

----
this.ultraTree1.Nodes[0].Nodes[1].Enabled = false;
----
