= 週末の展開表示

週末はデフォルトでは圧縮形式で表示されますが、全日形式で表示することもできます。

*Visual Basic の場合：*

----
Imports Infragistics.Win.UltraWinSchedule
...
Private Sub ShowWeekendsasFullDays_Load(ByVal sender As System.Object, _
  ByVal e As System.EventArgs) Handles MyBase.Load
	Me.UltraMonthViewSingle1.WeekendDisplayStyle = WeekendDisplayStyleEnum.Full
End Sub
----

*C# の場合：*

----
using Infragistics.Win.UltraWinSchedule;
...
private void ShowWeekendsasFullDays_Load(object sender, System.EventArgs e)
{
	this.ultraMonthViewSingle1.WeekendDisplayStyle = WeekendDisplayStyleEnum.Full;
}
----
