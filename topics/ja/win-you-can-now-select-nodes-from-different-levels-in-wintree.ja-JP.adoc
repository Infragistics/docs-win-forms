= WinTree で異なるレベルから複数のノードを選択できる

これまでのリリースでは、WinTree™ で同時にノードの選択は同じツリー レベルに限定されていました。今回のリリースではレベルの制限がなくなり、ツリーの複数レベルで必要なノードを選択できます。この機能を使用するために追加作業を行う必要はありません。ただし、異なるレベルのノードに基づいて pick:[win-forms="link:{ApiPlatform}win.ultrawintree{ApiVersion}~infragistics.win.ultrawintree.selectednodescollection.html[SelectedNodes]"]  コレクションを操作したい場合、コレクションを介して並べ替えてpick:[win-forms="link:{ApiPlatform}win.ultrawintree{ApiVersion}~infragistics.win.ultrawintree.ultratreenode~level.html[Level]"] に属すpick:[win-forms="link:{ApiPlatform}win.ultrawintree{ApiVersion}~infragistics.win.ultrawintree.ultratreenode.html[Node]"]  を識別する必要があります。
