= 放棄されたプロパティ

大幅な改善や小さな変更を行う必要があるので、製品のライフ サイクルには時期というものが常にあります。これらの変更はアプリケーションに影響を及ぼす場合があります。{ProductName} 2006 Volume 2 では、いくつかのプロパティを廃止する必要がありました。これは新しい Application Styling Framework™ の結果です。これらのプロパティにはデフォルト値を持つ必要がありました。したがって、Boolean のタイプのプロパティは、DefaultableBoolean のタイプでなければなりませんでした。以下の表は廃止されたプロパティと置き換えられる新しいプロパティのリストです。
