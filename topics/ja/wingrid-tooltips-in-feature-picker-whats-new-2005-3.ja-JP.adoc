= 機能ピッカーのツールチップ

デザイナで機能ピッカーのノードの上にマウスを置くと、ツールチップが表示されます。このツールチップには、項目の説明と、そのオプションを選択したときにどのプロパティがどの値に設定されるかを示す疑似コード スタイルのコード行が含まれます。
