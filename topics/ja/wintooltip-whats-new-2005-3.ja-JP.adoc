= WinToolTip 2005.3

Win アセンブリに新規の WinToolTip コンポーネントが追加されました。Microsoft ツールチップとまったく同様にこれらのコンポーネントをフォーム上に配置できますが、いくつか次のような機能が追加されています。

* バルーンチップのサポート
* 説明文が表示される前の時間とそれが見えている時間を制御
* コンポーネント上または個々のコントロール レベルで指定できるデフォルト可能な設定
* 外観設定
* ツールチップの位置の制御
