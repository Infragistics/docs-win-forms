== WinGanttView コントロールで印刷プレビュー使用する

[[_Ref317523196]]

=== 概要

WinPrintPreviewDialog で、任意の .NET PrintDocument を使用します。.NET PrintDocument は WinGanttViewPrintDocument™ を含んでおり、これを使用して WinGanttView™ コントロールの印刷をします。

[[_Ref317523203]]

=== プレビュー

以下のスクリーンショットは最終結果のプレビューです。

image::images/Print_Preview_with_WinGanttViewPrintDocument_1.png[]

[[_Ref317523211]]

=== 要件

この手順を行うには、以下が必要です。

* フォームに UltraPrintPreviewDailog、UltraGanttViewPrintDocument、UltraCalendarInfo、および UltraGanttView の各コントロールを追加します。
* フォームに UltraButton™ コントロールを追加し、Text プロパティを `‘Print Preview’`に設定します。
* UltraGanttView コントロールをデータにバインドし、タスクとタスク情報を表示します。詳細については、 link:winganttview-databinding-support-for-winganttview.html[DataBinding Support for WinGanttView] のトピックを参照してください。

[[_Ref317523217]]

=== 手順

以下のステップでは、WinGanttView、WinGanttViewPrintDocument および WinPrintPreviewDialog の各コントロールを設定して、GanttView コントロールを印刷プレビューします。

=== UltraGanttView コントロールを UltraGanttViewPrintDocument コンポーネントに割り当てます。

UltraGanttViewPrintDocument コンポーネントは link:{ApiPlatform}win.ultrawinganttview{ApiVersion}~infragistics.win.ultrawinganttview.ultraganttview_members.html[GanttView] プロパティを表示し、このプロパティにはフォーム上の UltraGanttView コントロールを割り当てておく必要があります。

*Visual Basic の場合:*

[source,vb]
----
Me.ultraGanttViewPrintDocument1.GanttView = Me.ultraGanttView1
----

*C# の場合:*

[source,csharp]
----
this.ultraGanttViewPrintDocument1.GanttView = this.ultraGanttView1;
----

=== UltraGanttViewPrintDocument を UltraPrintPreviewDialog コンポーネントに割り当てます。

UltraGanttViewPrintDocument コンポーネントは、UltraPrintPreviewDialog の link:{ApiPlatform}win.ultrawinprintpreviewdialog{ApiVersion}~infragistics.win.printing.ultraprintpreviewdialog~document.html[Document] プロパティに割り当ててください。

*Visual Basic の場合:*

[source,vb]
----
Me.ultraPrintPreviewDialog1.Document = Me.ultraGanttViewPrintDocument1
----

*C# の場合:*

[source,csharp]
----
this.ultraPrintPreviewDialog1.Document = this.ultraGanttViewPrintDocument1;
----

=== 印刷プレビュー ダイアログに WinGanttView コントロールを組み込みます。

Print Preview ボタンのクリック イベントにこのコードを記述します。

*Visual Basic の場合:*

[source,vb]
----
Me.ultraPrintPreviewDialog1.ShowDialog(Me)
----

*C# の場合:*

[source,csharp]
----
this.ultraPrintPreviewDialog1.ShowDialog(this);
----

=== アプリケーションの実行

アプリケーションを実行し、Print Preview ボタンをクリックします。印刷プレビュー ダイアログがポップアップし、GanttView コントロールを表示します。
