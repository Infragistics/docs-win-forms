= 固定（非スクロール）列の作成

固定（非スクロール）列を作成するには、pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.headerbase.html[Header]"]  オブジェクトの pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.headerbase~fixed.html[Fixed]"]  プロパティを使用します。

*Visual Basic の場合：*

----
Imports Infragistics.Win.UltraWinGrid
...
Me.UltraGrid1.DisplayLayout.UseFixedHeaders = True
----

*C# の場合：*

----
using Infragistics.Win.UltraWinGrid;
...
this.ultraGrid1.DisplayLayout.UseFixedHeaders = true;
----

デフォルトで表示されているヘッダの pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.headerbase~fixedheaderindicator.html[FixedHeaderIndicator]"]  をオン/オフにすることもできます。

*Visual Basic の場合：*

----
Imports Infragistics.Win.UltraWinGrid
...
Me.UltraGrid1.DisplayLayout.Bands(0).Columns(0).Header.FixedHeaderIndicator = _
  FixedHeaderIndicator.None
----

*C# の場合：*

----
using Infragistics.Win.UltraWinGrid;
...
this.ultraGrid1.DisplayLayout.Bands[0].Columns[0].Header.FixedHeaderIndicator = 
  FixedHeaderIndicator.None;
----

次の画像では、列 0（Address）は固定ヘッダを使用するように設定され、FixedHeaderIndicator は None に設定されています。
