= AppStylistRuntime コンポーネントについて

AppStylistRuntime コンポーネントによって、ご使用のアプリケーションに直接結びついた AppStylist™ のよりコンパクトなバージョンを実行できます。ShowRuntimeApplicationStyler メソッドを実行すると AppStylist のランタイム バージョンが表示されます。このコンポーネントを使用してオンザフライで UI を変更します。
