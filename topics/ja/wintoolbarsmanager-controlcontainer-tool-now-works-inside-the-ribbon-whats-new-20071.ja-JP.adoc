= ControlContainer がリボン内で機能する

2006 Volume 3 リリースでは、pick:[win-forms="link:{ApiPlatform}win.ultrawintoolbars{ApiVersion}~infragistics.win.ultrawintoolbars.controlcontainertool.html[ControlContainerTool]"]  は、リボンがサポートしていなかった複数のツールのひとつでした。ControlContainer ツールは、ツールバーの非常に重要な部分です。このツールは .NET コントロールをホストでき、ツールバーで完全な互換性を提供します。これで ControlContainer ツールは RibbonGroup で使用することができ、リボンで Windows Forms コントロールをホストすることができます。

デザインタイムとランタイムの両方のサポートで、ControlContainer ツールの追加は、他のツールをリボンに追加するのと同様に簡単です。Windows Forms コントロールのひとつまたはそれに対して .NET コントロールをインスタンス化すれば、そのコントロールに対して ControlContainer の pick:[win-forms="link:{ApiPlatform}win.ultrawintoolbars{ApiVersion}~infragistics.win.ultrawintoolbars.controlcontainertool~control.html[Control]"]  プロパティを設定する必要があります。
