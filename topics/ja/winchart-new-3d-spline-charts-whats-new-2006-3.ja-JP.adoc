= 新しい立体スプライン グラフ

link:chart-spline-chart-2d.html[平面スプライン]および link:chart-spline-area-chart-2d.html[平面スプライン エリア] グラフを補完するために、 link:chart-spline-chart-3d.html[立体スプライン]および link:chart-spline-area-chart-3d.html[立体スプライン エリア] グラフをそれぞれ提供しています。これらの新しい立体チャートは、もうひとつの次元（z 軸）を既存の平面チャートに追加し、その結果より視覚的に魅力的なチャートを作成することができます。
