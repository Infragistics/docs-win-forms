= WinGrid ClickCell イベント

WinGrid™ は pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragrid~clickcell_ev.html[ClickCell]"]  イベントを持つようになりました。既存の pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragrid~doubleclickcell_ev.html[DoubleClickCell]"]  イベントに非常に似ており、さまざまなシナリオに対応するために多くの種類のカスタム ロジックを実装できます。ClickCell イベント内では、イベント引数を介してクリックされたセルに簡単にアクセスできるので、その兄弟セルや値だけでなく、セル値、属する行にアクセスできます。
