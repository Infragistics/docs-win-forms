= 追加された GDI/GDI+ テキスト レンダリング プロパティ

{ProductName} 2005 Volume 3（CLR 2.0）リリースでは、テキストを GDI+ でレンダリングしようとすると、UseGDIPlusTextRenderer を True に設定しなければなりませんでした。True に設定されると、プロジェクト内のすべてのコントロールは GDI+ でレンダリングします。しかし、新しい TextRenderingMode プロパティが基本クラスに追加されたことにより、フォーム上のコントロールを指定して GDI+ レンダリングすることが可能となりました。
