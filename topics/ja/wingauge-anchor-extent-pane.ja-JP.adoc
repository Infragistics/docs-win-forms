= 範囲ペイン

[範囲] ペインによって、アンカーの位置を決定できます。

pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedleanchor~extent.html[範囲]"]  -- この値を 0 から 100 の整数に設定します。範囲が 0 に設定されると、アンカーはラジアル ゲージの中央に配置されます。値が大きくなると、アンカーは針の方向でゲージの端に向けて移動します。値が小さくなると、アンカーは反対方向に移動します。100 よりも大きい値を入力すると、アンカーはゲージに表示されません。
