= Fill Scene Graph イベント

新しい FillSceneGraph イベントは、ユーザー自身のカスタム レイヤーを作成することを不要にしました。このイベントは、Primitive のすべてが WinChart™ のシーン グラフに追加されているが、WinChart が描画される前に起動します。

FillSceneGraphEvent に渡される FillSceneGraphEventArgs は、ChartCore の複数の重要なメンバを公開します。
