= ドロップダウン コントロールでドロップダウンをリサイズ

WinComboEditor™、WinCombo™、WinDropDown™ および WinPopupControlContainer™ のようなドロップダウン コントロールでドロップダウンをリサイズするためのサポートが追加されました。ドロップダウンは、ハンドルを使用してリサイズできます。ハンドルはドロップダウンの一番下に垂直または水平に配置できます。ドロップダウンのリサイズされた高さと幅はアプリケーションのその実行に対して保存されます。このハンドルは通常の Infragistics® Appstyling と外観プロパティを使用したスタイリングをサポートしています。
