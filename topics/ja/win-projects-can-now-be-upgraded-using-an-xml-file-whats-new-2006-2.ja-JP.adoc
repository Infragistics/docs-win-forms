= XML ファイルを使用してプロジェクトをアップグレードできる

各 {ProductName} ボリューム リリースおよびパブリック ホット フィックスで提供されている新しい機能や特徴を活用するには、既存のプロジェクトを最新の {ProductName} バージョンにアップグレードする必要があります。多数のプロジェクトをアップグレードする必要がある人のために、弊社は 2006 Volume 2 リリースのアップグレード プロセスを改善しました。これで XML ファイルを単に作成し、この XML ファイルで指定したプロジェクトをアップグレードするためにプロジェクトアップグレード ユーティリティを起動するコマンドを実行することによって、一度にすべてのプロジェクトをアップグレードすることができます。
