= WinToolTipManager について

WinToolTipManager コントロールを使用すると、Windows Formsの任意のコントロールに "UltraToolTip" というエクステンダ プロパティを追加できます。UltraToolTip プロパティは、WinToolTipManager でコントロールを管理することを目的とします。これにより、マウスがコントロールの上に置かれたときにツールチップを自動的に表示できるようになります。

* 外観のサポート - アルファ ブレンディング、グラデーション、画像などを含め、完全な外観がサポートされます。
* BalloonTips -- ツールチップの隅を丸めて吹き出しのような外観にします。
