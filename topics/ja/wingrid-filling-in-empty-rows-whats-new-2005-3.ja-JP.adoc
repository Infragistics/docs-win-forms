= 空行の充てん

グリッドの残りの行を空の行で満たします。これは厳密に言うとルック アンド フィールの拡張であるので、エンド ユーザーはより優れたグリッドの外観が得られます。空行の機能によって、最後の行の後にある WinGrid 内の空の領域を空行で満たすことができます。これらの空行はアクティブ化できません。これらの行は UI インタラクティブです。
