= 外観の選択

このトピックはすべての WinEditor コントロールに適用されます。

いくつかのアプリケーションで整合性のある外観を作成するために、WinEditor コントロールのいずれかのルック アンド フィールを変更したい場合があります。pick:[win-forms="link:{ApiPlatform}win.ultrawineditors{ApiVersion}~infragistics.win.ultrawineditors.texteditorcontrolbase~displaystyle.html[DisplayStyle]"]  プロパティによって、OfficeXP、Office2000、VisualStudio2005、Office2007 およびのより多くのスタイルから選択することができます。以下のサンプル コードは、Office2007 のルック アンド フィールを使用するように DisplayStyle プロパティを設定する方法を示します。

*Visual Basic の場合：*

----
Imports Infragistics.Win
...
Private Sub Choose_the_WinEditors_Look_Load(ByVal sender As System.Object, _
  ByVal e As System.EventArgs) Handles MyBase.Load
	Me.UltraComboEditor1.DisplayStyle = EmbeddableElementDisplayStyle.Office2007
End Sub
----

*C# の場合：*

----
using Infragistics.Win;
...
private void Choose_the_WinEditors_Look_Load(object sender, EventArgs e)
{
	this.ultraComboEditor1.DisplayStyle = EmbeddableElementDisplayStyle.Office2007;
}
----
