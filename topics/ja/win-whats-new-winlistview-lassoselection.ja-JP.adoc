= WinListView LassoSelection

Windows Explorer のユーザーに馴染みのあるひとつの機能は「投げ縄選択」で、これが WinListView™ コントロールに今回追加されました。「投げ縄選択」という語句は、ビューでひとつ以上の項目の周囲をドラッグし、その結果選択された状態にする選択矩形のことを指します。この機能によって、WinListView で値の範囲の選択が可能になります。
