= 合成チャート

新しい ChartType が Chart に追加されました。これによって、新しいチャートを作成するために既存のチャートと合成することが可能になります。この新しいチャート タイプと共に、この機能を可能にするために複数の変更が行われました。

新しいコレクションがチャートに追加されました。

* Series コレクションは、複数のデータ系列を含むことができます。各系列は複数のデータ点を含むことができます。
* ChartAreas コレクションは、チャート レイヤを追加できる個別の領域を含むことができます。各 ChartArea にひとつまたは複数のチャート レイヤを追加できます。
* ChartLayers コレクションによって、複数のチャート タイプを表示に追加することができます。
