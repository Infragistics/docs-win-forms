= WinGanttViewPrintDocument について

WinGanttViewPrintDocument™ は、WinGanttView™ を簡単に印刷またはプレビューするための Windows Forms コンポーネントです。以下に WinGanttViewPrintDocument で使用できる機能の一部について説明します。

* プリントプレビュー － WinGanttViewPrintDocument コンポーネントは印刷前に WinGanttView ドキュメントをプレビューするために WinPrintPreviewDialog コンポーネントを使用することができます。詳細は、 link:winganttviewprintdocument-print-preview-with-winganttviewprintdocument.html[WinGanttViewPrintDocument と共に WinPrintPreviewDialog を使用する]を参照してください。
* ヘッダー / フッター -- WinGanttView コントロールとともにカスタムのヘッダーとフッターを印刷するように指定できます。
* InitializeGanttView Event - スクリーン上に現れる WinGanttView コントロールに影響を与えることなく、印刷のために使われる WinGanttView のプロパティの変更ができます。
