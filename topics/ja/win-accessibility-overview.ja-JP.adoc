= アクセシビリティ概要

Windows Forms でビルドされたアプリケーションは、障害を持つ人に優れた対応力があります。 Windows® Narrator はドキュメント全文を朗読できます。 Windows Magnifier は、マウス カーソルで拡大して焦点を合わせることができます。 これらの各アプリケーションは、.NET Framework の AccessibleObject クラスのため、ユニークな方法で他のアプリケーションと相互作用します。 AccessibleObject を通して、アプリケーションは Narrator および Magnifier を開き、画面の各オブジェクト内に何が含まれているかを確認することができます。 AccessibleObject の他のインスタンスは、次に来る補助的な技術の小さい UI 要素を表すために作成されます。

コントロールのいくつかは AccessibleObject を使用し、Narrator および Magnifier などのアプリケーションにアクセスできます。 この技術はビジュアル コントロールおよびコンポーネントに限って有益です。一部の {ProductName} コントロールおよびコンポーネントは視覚的ではないので、アクセシビリティは実装されません。
