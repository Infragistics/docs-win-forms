== Infragistics NuGet パッケージ

Infragistics {ProductName} は、製品の各コントロールおよびコンポーネントの NuGet パッケージを提供します。以下の表は、コントロールを使用するために {PlatformName} アプリケーションに追加するパッケージを参照できます。

NuGet パッケージを使用する方法は 2 つあります。パッケージ ファイルをコンピューターのディレクトリに保存できてそのディレクトリへ参照する link:nuget-feeds.html#adding_local_package[パッケージ ソースを追加]またはライセンス付きの link:nuget-feeds.html#adding_infragistics_feed[Infragistics NuGet フィード] を使用できます。ライセンス付きのフィードを使用するには、有効な製品キーに紐づけされた Infragistics アカウントが必要です。

- 2019.1 リリースでは、以下にリストされている NuGet パッケージが .NET Core 3.0 環境でサポートされています。
- 2020.2 リリース以降、.NET Core 3.0 はサポートされなくなり、.NET Core 3.1 および .NET 5.0 がサポートされます。
- 2021.2 リリース以降、.NET 6.0 がサポートされます。
- 2022.2 リリース以降、.NET 7.0 がサポートされます。
- 2023.2 リリース以降、.NET 8.0 がサポートされます。
- 2024.2 リリース以降、.NET 9.0 がサポートされます。NET 3.1、.NET 5、.NET 7 はサポートされなくなりました。
- 2025.2 リリース以降、.NET 10.0 がサポートされます。.NET 6.0 はサポートされなくなりました。
