= 範囲ペイン

[範囲] ペインによって、スケールの太さと位置を決定できます。ラジアル ゲージではこの位置は中心から測定されますが、リニア ゲージではこの位置はゲージの下側から測定されます。

pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugescale~innerextentstart.html[内側の開始]"]  -- この値を 0 から 100 の整数に設定します。この値は、スケールの内側の縁の開始点の位置を決定します。

pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugescale~innerextentend.html[内側の終了]"]  -- この値を 0 から 100 の整数に設定します。この値は、スケールの内側の縁の終了点の位置を決定します。

pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugescale~outerextent.html[外側]"]  -- この値を 0 から 100 の整数に設定します。この値は、測定点からスケールの外側の縁上のあらゆる点への距離を決定します。
