= 新しい WinTimeSpanEditor コントロール

Infragistics® Windows Forms コントロールの本リリースでは、新しい WinTimeSpanEditor™ コントロールを提供します。TimeSpan データ タイプを編集するために、このエディターを使用できるようになりました。WinTimeSpanEditor コントロールは、IProvidesEmbeddableEditor インターフェイスを実装しているので、WinGrid™、WinTree、および WinSchedule コントロールのような Infragistics コントロール内に組み込むことができます。このコントロールには、タイム スパン値を増減するために使用できる組み込みのスピン ボタンがあります。
