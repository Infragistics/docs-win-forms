= バージョンフリー アセンブリ名

2010 Volume 2 の本リリースから、製品インストールとともに zip フォーマットで使用できるバージョンフリー アセンブリを提供します。バージョンフリー アセンブリには、名前に末尾バージョン番号および CLR 識別番号はありません。たとえば、{ApiPlatform}Shared.v10.2.dll は Infragistics.Shared.dll として読み込みます。CLR2 と CLR 3 の両バージョン（{ApiPlatform}Documents.Excel.v10.2 と Infragistics3.Documents.Excel.v10.2 のような）を持つアセンブリの場合、バージョンフリー アセンブリは最新の CLR バージョンと一致します。したがって、バージョンフリーの Excel アセンブリは Infragistics.Documents.Excel.dll であり、CLR3 バージョンのためにビルドされたアセンブリです。バージョンフリー アセンブリを使用すれば、アセンブリ バインディング リダイレクトを使用して、アプリケーションを再ビルドすることなく、ひとつのバージョンから別のバージョンに簡単にアップグレードできます。
