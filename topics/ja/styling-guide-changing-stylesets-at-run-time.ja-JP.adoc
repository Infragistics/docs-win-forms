= ランタイムにスタイルセットを変更

Infragistics AppStylist™ で作成した異なるスタイルをエンド ユーザーに選択させたい場合、スタイルセットを変更する必要があります。デザイン タイムまたはランタイムでスタイルセットを変更できます。デザイン タイムにスタイルセットを変更するには、アプリケーション スタイリングをサポートする Infragistics コントロールの pick:[win-forms="link:{ApiPlatform}win{ApiVersion}~infragistics.win.ultracontrolbase~stylesetname.html[StyleSetName]"]  プロパティを使用します。ラン タイムでは、スタイルセットを変更するには以下のコードを使用します。

*Visual Basic の場合：*

----
Me.ultraMonthViewSingle.StyleSetName = "StyleSetBlue"
----

*C# の場合：*

----
this.ultraMonthViewSingle.StyleSetName = "StyleSetBlue";
----
