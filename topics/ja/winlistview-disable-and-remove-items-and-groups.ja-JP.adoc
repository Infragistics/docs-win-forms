= 項目およびグループの無効化および削除

このトピックは、Item オブジェクトの pick:[win-forms="link:{ApiPlatform}win.ultrawinlistview{ApiVersion}~infragistics.win.ultrawinlistview.ultralistviewitem~enabled.html[Enabled]"]  プロパティを False に設定することによって、WinListView™ に追加された項目とグループを無効にできる方法を示します。Items または Groups オブジェクトの pick:[win-forms="link:{ApiPlatform}win.ultrawinlistview{ApiVersion}~infragistics.win.ultrawinlistview.ultralistviewitemscollection~remove.html[Remove]"]  メソッドと pick:[win-forms="link:{ApiPlatform}win.ultrawinlistview{ApiVersion}~infragistics.win.ultrawinlistview.ultralistviewitemscollection~removeat.html[RemoveAt]"]  メソッドを使用して、WinListView から項目またはグループ全体を削除できる方法も説明します。

*Visual Basic の場合：*

----
' 項目を無効にします
 Me.ultraListView2.Groups[0].Items[1].Enabled = False
' グループを無効にします
Me.ultraListView1.Groups[1].Enabled = False
----

*C# の場合：*

----
// 項目を無効にします
  this.ultraListView2.Groups[0].Items[1].Enabled = false;
// グループを無効にします
  this.ultraListView1.Groups[1].Enabled = false;
----

*Visual Basic の場合：*

----
' WinListView で特定の項目をグループから削除します 
Me.ultraListView1.Items.Remove(Me.ultraListView1.Groups(0).Items(1)) 
' 指定された順序位置（項目が作成された位置であり、表示位置ではない）から項目を削除します 
Me.ultraListView1.Items.RemoveAt(3) 
' グループを削除します 
Me.ultraListView1.Groups.Remove(Me.ultraListView1.Groups(1)) 
'順序位置を指定してグループを削除します 
Me.ultraListView1.Groups.RemoveAt(0)
----

*C# の場合：*

----
// WinListView で特定の項目をグループから削除します
 this.ultraListView1.Items.Remove(this.ultraListView1.Groups[0].Items[1]);
// 指定された順序位置（項目が作成された位置であり、表示位置ではない）から項目を削除します
 this.ultraListView1.Items.RemoveAt(3);
// グループを削除します
this.ultraListView1.Groups.Remove(this.ultraListView1.Groups[1]);
//順序位置を指定してグループを削除します
this.ultraListView1.Groups.RemoveAt(0);
----
