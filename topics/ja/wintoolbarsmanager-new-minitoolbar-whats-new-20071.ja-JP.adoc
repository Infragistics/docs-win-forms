= 新しい MiniToolbar

Microsoft® Word 2007 は、現在選択しているテキストの近くにツールバーをフロートし、ツールのフォーマッティング タイプを提供する新しいタイプを導入しています。2007 Volume 1 リリースでは、Infragistics は MiniToolbar も提供しています。

MiniToolbar は、コンテキスト メニュー付きまたはなしでマウス ポインタの上に表示できる別のタイプのツールバーです。MiniToolbar にツールを追加するのは、他のツールバーと同じ手順です。ツールを作成し（または既存のツールを使用）、それを pick:[win-forms="link:{ApiPlatform}win.ultrawintoolbars{ApiVersion}~infragistics.win.ultrawintoolbars.minitoolbar~tools.html[Tools]"]  コレクションに追加し、MiniToolbar の Tools コレクションに追加します。MiniToolbar を表示したいツールの行数の制御も行います。

MiniToolbar が完全に不透明になるポイントにマウスを移動するまで右クリックで MiniToolbar は半透明で表示されます。マウスを MiniToolbar から遠ざけると、より透明になります。MiniToolbar から一定の距離まで移動すると完全に消えます。
