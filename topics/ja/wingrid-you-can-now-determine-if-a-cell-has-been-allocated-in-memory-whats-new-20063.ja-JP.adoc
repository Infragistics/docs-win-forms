= セルがメモリに割り当てられたかどうかを判断できるようになった

エンド ユーザーが WinGrid のセルを修正すると、WinGrid はエンド ユーザーの修正に基づいてより多くのメモリを割り当てます。Row オブジェクトの新しい pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridrow~hascell.html[HasCell]"]  メソッドを使用して、WinGrid がセルを割り当てるかどうかを判断できます。pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridrow~deallocatecells.html[DeallocateCells]"]  メソッドを使用して、メモリから割り当てられたセルを削除できます。
