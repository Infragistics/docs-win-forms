= 新しいリソース プレビュー

== 複数の領域でリソースをプレビューすることができるようになった

これまで、リソースをプレビューしたい場合には、[プレビュー] ペインをオンにする必要がありました。これからはスタイル エクスプローラの [リソース] タブと UI ロール エディタの [リソース] ペインの両方でリソースを簡単に見ることができます。
