= DataGridView での WinCalcManager の使用

Visual Studio 2003（.NET Framework 1.1）以前では、Microsoft の組み込みコントロールである DataGrid で WinCalcManager はサポートされていませんでした。Visual Studio 2005（.NET Framework 2.0）では、DataGridView コントロール内で WinCalcManager が完全にサポートされています。
