= 放射型グラデーション ブラシ エレメント

ゲージのプロパティの 2 色を円形に相互に混ぜたい場合には、ブラシ タイプを RadialGradient に適用します。RadialGradient ブラシ エレメントを選択すると、中心点とフォーカス スケールだけでなく、中心と周囲の色を指定できます。以下を使用して放射型グラデーション ブラシ エレメントをゲージのプロパティに適用することができます。

* link:wingauge-apply-brush-elements-using-the-gauge-designer.html[ゲージ デザイナ]
* link:wingauge-apply-brush-elements-at-design-time.html[ゲージ コレクション エディタ]
* link:wingauge-apply-brush-elements-at-run-time.html[コード]

以下のスクリーンショットは、以下の設定で目盛盤の BrushElement を RadialGradientBrushElement に設定した状態のラジアル ゲージを表示します。

* 中心の色 -- Peach Puff
* 周囲の色 -- Sandy Brown
* 中心点 -- 25, 25
* フォーカス ポイント -- 0, 0
* 中心点の単位 -- パーセント
