= WinSplitter について

WinSplitter™ コントロールによってエンドユーザーは、ランタイムに WinSplitter の端にドックされるコントロールをリサイズできます。 エンドユーザーが WinSplitter の上にマウスを移動すると、カーソルは WinSplitter にドックされるコントロールがリサイズできることを示すために変更されます。 フォームの Top/Bottom または Left/Right に WinSplitter をドックできるので、コンテンツは水平方向または垂直方向にリサイズできます。

WinSplitter コントロールの機能の一部は以下のとおりです:

* *コンテンツの縮小/復元：* 縮小/復元ボタンは、フォームでコンテンツを縮小または復元するために WinSplitter コントロールで使用できます。
* *スタイリング：* WinSplitter は、Appearance 設定および AppStyling からスタイルできます。 縮小/復元ボタンは、さまざまなスタイル プロパティでスタイルすることもできます。
