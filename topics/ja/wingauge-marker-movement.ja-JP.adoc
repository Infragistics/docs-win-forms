= マーカーの移動

スケール上の異なる値を指し示すためにマーカーの値を構成できます。

これによって、エンド ユーザーはゲージを操作できます。エンド ユーザーはラジアル ゲージで針をドラッグしてスケール上の異なる値を指し示すことができます。ラジアル ゲージの針マーカーは、以下の方法で移動できます。

* 手動 -- エンド ユーザーが手動で針を移動できるように構成できます。これによって、針がスケールで指し示している値をエンド ユーザーが変更することが可能となります。
* 自動 -- スケールで特定の時間の間隔と距離でゲージ上の針を自動的に移動するように構成できます。

リニア ゲージでは、マーカーは自動でしか動かすことができません。
