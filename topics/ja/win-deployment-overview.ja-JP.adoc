= 配置の概要

{ProductName} ツールセットのツールを含むアプリケーションを配備する最初の手順として、配備パッケージの一部として含む必要があるリソースを決定しなければなりません。以下のリンクをクリックすると、各要素が必要とするリソースを学習します。

$$.NET$$ アプリケーションを配備する時、実行ファイルに加えて特定のファイルを配布する必要があります。アプリケーションに 1 つ以上の Infragistics 要素を含む場合、アプリケーションの一部として 1 つ以上の Infragistics アセンブリ ファイルを配布または配備する必要があります。このトピックでは、再配布する必要があるファイルの概要を述べます。

[NOTE]
====
注: 以降の .NET Framework バージョン番号 (Infragistics3- など) で、このトピックにリストされたファイルは、再配布可能です。このトピックに記載されていないファイルは、再配布不可能と見なしてください。Infragistics 製品に含まれているファイルで一覧にないファイルは配布できません。Infagistics 製品のライセンスのない他のマシンへのコピー、移動、共有が禁止されています。
====

== すべてのコントロール/コンポーネントで必要なアセンブリ

各コントロール/コンポーネントにリストされる個々のアセンブリ ファイルに加えて、{ProductName} コントロール/コンポーネントを配備するには次のファイルの再配布も必要です。

[options="header", cols="a,a"]
|====
|ファイル名|説明

|{ApiPlatform}Shared.v{ProductVersion}.dll
|$$.NET$$ 用の Infragistics Shared サポート アセンブリ

|{ApiPlatform}Win.v{ProductVersion}.dll
|$$.NET$$ 用の {ProductName} サポート アセンブリ

|====

以下のコントロールおよびコンポーネントは Win または Shared アセンブリを必要としません。

* Infragistics Syntax Parsing Engine
* Infragistics ドキュメント エンジン
* Infragistics Excel エンジン
* Infragistics Word ライブラリ
* WinChart
* WinGauge (Shared は*必要*ですが、Win では*不要* )

== インク対応アプリケーションで必要なアセンブリ

TabletPC プラットフォームでのインクの使用をサポートするアプリケーションの場合には、インク機能を有効にするために次のアセンブリの配布も必要です。

[options="header", cols="a,a"]
|====
|ファイル名|説明

|{ApiPlatform}Win.UltraWinInkProvider.Ink17.v{ProductVersion}.dll
|Tablet PC Platform SDK *v1.7* に使用する Infragistics Ink Provider アセンブリです。

|====
