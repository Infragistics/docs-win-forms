= WinGridPrintDocument 総ページ数の印刷

この新しい機能によって、WinGridPrintDocument™ コンポーネントは追加の置き換え文字列 (<##> または <Total Pages>) を使用することで印刷される物理的なページ総数を印刷することが可能となります。以前は現在の物理的なページ数しか印刷できませんでした (例、"Page<#>")。
