= PLF および WinGrid パフォーマンス向上

本リリースでは、{ProductName} アプリケーションのパフォーマンスを大幅に向上する裏の調整を行いました。{ProductName} の中心である link:win-plf-overview.html[Presentation Layer Framework™ (PLF)] は、多くのパフォーマンスの向上を得て、すべてのコントロールおよびコンポーネントの全体的な効率および速度を向上しています。また、WinGrid™ コントロールは、複数の個々の拡張を得た結果、データ クイッカーをロードすることとなりました。最終結果として、コードを 1 行も書くことなくパフォーマンス ゲインを利用できます。
