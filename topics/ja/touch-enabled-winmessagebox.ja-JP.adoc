= タッチ対応の WinMessageBox

== トピックの概要

=== 目的

このトピックは、 _Infragistics_   のタッチ対応 WinMessageBox™ コントロール要素を紹介します。

== _WinMessageBox_   のタッチ対応要素

=== タッチ対応要素

以下のスクリーンショットは、 _WinMessageBox_   コントロールのタッチ対応要素を示します。タッチ要素は、タッチ対応モードで若干大きく表示されます。

.注:
[NOTE]
====
link:{ApiPlatform}win{ApiVersion}~infragistics.win.touch.ultratouchprovider_members.html[UltraTouchProvider] のタッチ対応設定の状態はキャッシュされ、メッセージ ボックスの表示中に使用されます。 link:{ApiPlatform}win{ApiVersion}~infragistics.win.touch.ultratouchprovider_members.html[UltraTouchProvider] の `Enabled` プロパティを切り替えても、すでに表示されたメッセージ ボックスに大きな影響はありません。
