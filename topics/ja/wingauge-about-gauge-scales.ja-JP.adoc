= ゲージ スケールについて

Gauge コントロールの pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgauge~scales.html[Scales]"]  プロパティは、リニア ゲージまたはラジアル ゲージに値のレンジを表示するために使用されます。スケールは、ゲージの開始および終了の測定点を示すために使用されます。
