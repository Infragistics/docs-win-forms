= WinActivityIndicator について

アプリケーション開発で、シーンの背後でプロセスが実行中であることをエンドユーザーに通知する方法を必要とする場合があります。 WinActivityIndicator™ は何かが起こっていることを示すために連続するアニメーションを使用してこの機能を果たします。 これはエンドユーザーに提供できるインタラクティブ性のレベルを高めます。

WinActivityIndicator は、Appearance オブジェクトと AppStyling を使用して簡単にカスタマイズおよびスタイルできる軽量のスタンドアローンのコントロールです。

以下は WinActivityIndicator の主な機能です。

* さまざまな色とグラデーションを使用してスタイルできます。
* マーキー スタイルのアニメーション。
* 連続（一方向）またはバウンスバック アニメーションを付けることができます。
* アニメーション速度を変更できます。
