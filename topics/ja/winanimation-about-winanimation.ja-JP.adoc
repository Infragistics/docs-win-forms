= WinAnimation コントロールについて

AnimationControl は、Microsoft Common Controls のアニメーション クラスのラッパーです。このコントロールは、AVI ファイル、またはストック AVI ファイルの 1 つを再生する機能を提供します。

[NOTE]
====
*注:* Animation Control には、アンマネージ コード権限が必要です。フォームに要素のインスタンスが含まれていて、アセンブリにアンマネージ コードの権限がない場合は、.NET Framework によってセキュリティ例外が生成され、このフォームは表示されません。
====
