= ColumnStyle を使用してエディタの割り当て

各 WinGrid™ pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridcolumn.html[Column]"]  は埋め込みエディタを使用して、エンド ユーザーがセルの値を変更することを可能にします。pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.columnstyle.html[ColumnStyle]"]  プロパティを設定することで埋め込みエディタのタイプを変更できます。詳細については、 link:wingrid-embeddable-editors.html[埋め込みエディタ]を参照してください。

以下のコード例は、ドロップダウン カレンダーで列を編集する方法を示します。

*Visual Basic の場合：*

[source,vb]
----
Me.UltraGrid1.DisplayLayout.Bands(0).Columns(0).Style = Infragistics.Win.UltraWinGrid.ColumnStyle.DropDownCalendar

----

*C# の場合：*

[source,csharp]
----
this.ultraGrid1.DisplayLayout.Bands[0].Columns[0].Style = Infragistics.Win.UltraWinGrid.ColumnStyle.DropDownCalendar;
----
