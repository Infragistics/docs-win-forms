= WinDropDown について

WinDropDown コントロールは、WinGrid と使用可能な Windows Formsのコントロールのひとつで、グリッド セル内に複数列のドロップダウン リストを提供します。WinGrid コントロールは、単一列の単純なドロップダウン リストをサポートしています。WinDropDown は、WinGrid と同じ基本クラスから派生し、そのほとんどの機能を継承しています（ソート、フィルタリング、外観など）。グリッドの列またはセルの pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridcomboeditor~valuelist.html[ValueList]"]  プロパティを使用して WinDropDown コントロールをグリッドに関連付けることができます。
