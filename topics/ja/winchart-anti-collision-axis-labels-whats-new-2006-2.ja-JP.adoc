= 衝突防止軸ラベル

これまでは、チャート、x 軸、および y 軸のラベルのサイズとコンテンツによって、衝突したり重なったりすることがありました。たとえば、以下の柱状グラフでは、x 軸のラベルが水平方向のシリーズ ラベルで切り取られています。

image::Images/WinChart_Anti_Collision_Axis_Labels_Whats_New_2006_2_01.png[]

2006 Volume 2 リリースから開始しましたが、AxisLabelAppearanceBase.Layout プロパティが Chart コントロールに追加されました。この機能によって、チャート ラベルが互いに衝突しないようにさまざまな衝突防止手段を使用するように各チャート軸を構成できます。これらの手段にはフォントのスケーリング、テキストの折り返し、回転、ラベルを交互に配置、テキストの切り取りが挙げられます。さらに、ラベルが衝突しないように組み込みのインテリジェントな衝突防止レイアウト アルゴリズムを使用するようにチャート軸を構成することができます。またはラベルが衝突しないようにチャートが使用する衝突防止方法を手動で定義することができます。

以下の柱状グラフは、新しい衝突防止軸ラベル機能が有効になっている例です。シリーズ レベルとともに、長い ｘ 軸ラベルを表示する十分な余裕があることを保証するために自動レイアウトがこの機能を変更することに注意してください。軸ラベルのフォントをスケールし、軸ラベルのテキストを折り返し、シリーズ ラベルのテキストを切り取りました。
