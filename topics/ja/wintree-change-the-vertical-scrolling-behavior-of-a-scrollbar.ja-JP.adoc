= スクロールバーの垂直スクロール動作の変更

WinTree の垂直スクロールの動作を制御する 2 つの設定があります。これらの設定は pick:[win-forms="link:{ApiPlatform}win.ultrawintree{ApiVersion}~infragistics.win.ultrawintree.ultratree~scrollbounds.html[ScrollBounds]"] プロパティによって決定されます。ScrollBounds プロパティを ScrollToLastItem（デフォルト設定）に設定すると、最後のノードのみが表示された状態になるまでツリーが下にスクロールされます。ScrollBounds プロパティを ScrollToFill に設定すると、最後のノードがコントロールの表示可能領域の下に表示されるようにツリーをスクロールします。

次のコードは、ScrollBounds プロパティを ScrollToFill に設定する方法を示します。

*Visual Basic の場合：*

----
Imports Infragistics.Win.UltraWinTree
...
Private Sub HowtoChangetheVerticalScrollingBehavioroftheScrollbar_Load(ByVal sender As _
  System.Object, ByVal e As System.EventArgs) Handles MyBase.Load
	Me.UltraTree1.ScrollBounds = ScrollBounds.ScrollToFill
End Sub
----

*C# の場合：*

----
using Infragistics.Win.UltraWinTree;
...
private void HowtoChangetheVerticalScrollingBehavioroftheScrollbar_Load(
  object sender, System.EventArgs e)
{
	this.ultraTree1.ScrollBounds = ScrollBounds.ScrollToFill;
}
----
