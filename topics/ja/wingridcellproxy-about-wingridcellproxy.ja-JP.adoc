= WinGridCellProxy について

WinGridCellProxy™ は、WinGridRowEditTemplate™ コントロールのエディタとして動作する一意のコントロールです。2 つのコントロールの一意のデザインのため、WinGridCellProxy は WinGridRowEditTemplate コントロール内だけで機能できます。WinGridCellProxy の link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridcellproxy~columnkey.html[ColumnKey] プロパティを基本データの対応する列のキーに設定することで、WinGridCellProxy は編集テンプレートが表示する行に列データを表示します。WinGridCellProxy は多目的の組み込み可能なエディタとして機能し、列のデータを表示する特定の組み込み可能なエディタ、つまり「セル プロキシー」という用語の役割を果たします。
