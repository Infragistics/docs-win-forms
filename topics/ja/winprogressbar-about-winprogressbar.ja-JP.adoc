= WinProgressBar について

WinProgressBar コントロールには、進行状況の表示を制御するさまざまなプロパティがあります。プログレスバーの色は、現在のパーセント値によって変えることができます。あるいは、プログレスバーを塗りつぶした画像として表示することも可能です。

* 外観のサポート - アルファ ブレンディング、グラデーション、画像などを含め、完全な外観がサポートされます。
* スタイル -- プログレスバーの増加を表すスタイルを指定できます（セグメントごととするか、連続イメージとするかなど）。
