= スピン ボタンの表示

このトピックはすべての WinEditor コントロールに適用されます。

デフォルトで、WinEditors はスピン ボタンを表示しません。ただし、pick:[win-forms="link:{ApiPlatform}win{ApiVersion}~infragistics.win.spinbuttondisplaystyle.html[SpinButtonDisplayStyle]"]  プロパティを設定することによって、スピン ボタンを表示できます。これによってエンド ユーザーはマウスを使用して、スピン可能なセクションの現在の値を増やしたり減らしたりすることができます。pick:[win-forms="link:{ApiPlatform}win.ultrawineditors{ApiVersion}~infragistics.win.ultrawineditors.ultranumericeditorbase~spinbuttonalignment.html[SpinButtonAlignment]"]  プロパティはスピン ボタンがコントロールの左と左のいずれに表示するのかを決定します。pick:[win-forms="link:{ApiPlatform}win{ApiVersion}~infragistics.win.editorwithmask~spinwrap.html[SpinWrap]"]  プロパティは、スピン　ボタンが最小値/最大値に基づいてスピン可能なセクションの値をラップするかどうかを決定します。
