== Grouped Rows でホット トラッキングを有効化

[[_Ref341518673]]

=== 概要とプレビュー

`HotTracking` は、マウス ポイントが要素の上をパスする場合に UI 要素の外観を向上します。この場合、UI 要素が行、および link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridoverride~groupbyrowhottrackappearance.html[GroupByRowHotTrackAppearance] プロパティの設定に基づいてグループ化 され、結果を強調表示して表示します。 _WinGrid™_   は、 link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridoverride_members.html[UltraGridOverride] オブジェクトの link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridoverride~groupbyrowhottrackappearance.html[GroupByRowHotTrackAppearance] プロパティについて紹介します。このプロパティは、グループ化された行で `HotTracked` 項目の外観を定義できます。

image::images/WinGrid_HotTracking_Grouped_Rows_1.png[]

[[_Ref341518679]]

=== プロパティ設定

`HotTracking` の外観をコードで設定します。

*C# の場合:*

[source,csharp]
----
ultraGrid1.DisplayLayout.Override.GroupByRowHotTrackAppearance.BackColor = Color.LightSkyBlue;
----

*Visual Basic の場合:*

[source,vb]
----
ultraGrid1.DisplayLayout.Override.GroupByRowHotTrackAppearance.BackColor = Color.LightSkyBlue
----
