= 平面および立体ピラミッド チャート

2006 Volume 2 リリースで追加されている新しいチャートの中に平面および立体ピラミッド チャートがあります。ピラミッド チャートは、全体に対する数量として部分を表示する劇的な方法を提供します。アプリケーションでピラミッド チャートを使用すれば、ピラミッド全体とセグメントの高さを比較することによって、エンド ユーザーは素早く各セグメントの重要性を判断できます。
