= Visual Studio .NET 2005 スタイルのドッキング

DragWindowStyle 列挙体の新しい LayeredWindowWithIndicators 値を使用すると、ドッキング可能なペインまたはグループをドラッグしたときに、WinDockManager によって Visual Studio 2005 スタイルのドッキング インジケータが表示されます。ドラッグ中にマウスをドッキング インジケータ（たとえば上部ドッキング領域）の上に置くと、半透明のウィンドウが現れます。ドッキング インジケータをクリックすると、ドラッグされたペインまたはグループはクリックしたドッキング インジケータの対応する位置にドッキングされます。
