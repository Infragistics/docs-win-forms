= 幅と範囲ペイン

[幅と範囲] ペインによって、リニア ゲージまたはラジアル ゲージ上で針マーカーの形状と位置を設定することができます。

幅 -- 以下の値を指定することによって、針の形状をカスタマイズします。

* pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedle~startwidth.html[開始]"]  -- この値を 0 から 100 の整数に設定します。この値は針マーカーの開始点の幅を設定します。
* pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedle~endwidth.html[中間]"]  -- この値を 0 から 100 の整数に設定します。この値は針マーカーの中間点の幅を設定します。
* pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedle~endwidth.html[終了]"]  -- この値を 0 から 100 の整数に設定します。この値は針マーカーの終了点の幅を設定します。

範囲 -- このセクションは針マーカーの位置を決定します。ラジアル ゲージではこの位置は中心から測定されますが、リニア ゲージではこの位置はゲージの下側から測定されます。

* pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedle~startextent.html[開始]"]  -- この値を 0 から 100 の整数に設定します。この値は針マーカーの開始点の位置を設定します。
* pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedle~midextent.html[中間]"]  -- この値を 0 から 100 の整数に設定します。この値は針マーカーの中間点の位置を設定します。
* pick:[win-forms="link:{ApiPlatform}win.ultrawingauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedle~endextent.html[終了]"]  -- この値を 0 から 100 の整数に設定します。この値は針マーカーの終了点の位置を設定します。
