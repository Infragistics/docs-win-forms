= セル値のインプレース編集

セル値をインプレースで編集できるようになりました。また、セルのインプレース編集に加えて、コントロールがデータバインドされている場合、編集モード セッションの結果を対応するデータ フィールドに書き戻す機能もサポートされました。

WinGrid コントロールでは列で使用される埋め込みエディタが列のデータ型に基づいて自動的に割り当てられるため、WinTree も同じように動作するよう変更されました。ただし、下位互換性を維持するため、列または列に関連付けられた ColumnSet（または ColumnSettings）の AllowCellEdit プロパティをデフォルト以外の値に設定しない限り、エディタは自動的に選択されません。

* 既存の埋め込みエディタ アーキテクチャを利用してセルをインプレースで編集するためのサポートが追加されました。
* キーボードを使用してセル間を移動するための基本サポートが追加されました。
* エンド ユーザーが編集モードセッションで入力した値を対応するデータ フィールドにコミットするためのサポートが追加されました。
* 列のデータ型に基づいて最も適切なエディタを自動的に選択するためのサポートが追加されました。
