= カスタム カラーと Office 2007 スタイルのカラー スキームをブレンド

2006 Volume 3 では、弊社は Microsoft® Office 2007 スタイルのルック アンド フィール全般をリリースしました。このルック アンド フィールを達成するために、青、黒、およびシルバーの 3 つのカラー スキームを採用しました。65,000 以上から 3 色しか使用できないということは若干の制約があります。したがって、2007 Volume 1 では、3 つの既存のカラー スキームと任意の色をブレンドする機能を提供しています。pick:[win-forms="link:{ApiPlatform}win{ApiVersion}~infragistics.win.office2007colortable.html[Office2007ColorTable]"]  オブジェクトの新しい pick:[win-forms="link:{ApiPlatform}win{ApiVersion}~infragistics.win.office2007colortable~customblendcolor.html[CustomBlendColor]"]  プロパティを使用して、青、黒、またはシルバーのカラー スキームとシステム カラーをブレンドできます。

以下のコードは、Office 2007 カラー スキームを黒に設定し、赤と黒のスキームをブレンドします。

[NOTE]
====
*注：* これらの変更を確認するためには、コントロールの表示スタイルを Office 2007 に設定する必要があります。
====

image::images/Win_Blend_a_Custom_Color_with_Office_2007_Style_Color_Schemes_01.png[]

*Visual Basic の場合：*

----
Imports Infragistics.Win
...
Office2007ColorTable.ColorScheme = Office2007ColorScheme.Black
Office2007ColorTable.CustomBlendColor = Color.Red
----

*C# の場合：*

----
using Infragistics.Win;
...
Office2007ColorTable.ColorScheme = Office2007ColorScheme.Black;
Office2007ColorTable.CustomBlendColor = Color.Red;
----
