== 概要

=== WinTree 印刷の概要

_WinTree_   印刷機能は、印刷プレビューや印刷の機能性を提供することを目的としています。 _WinTree_   は、 _標準_  、 _OutlookExpress_  、 _グリッド_  および _フリーフォーム_  など、サポートされたビュー スタイルで表示または印刷することができます。

以下のスクリーンショットは、 _WinTree_   ( _OutlookExpress_   ビュー スタイル) の印刷プレビューを示しています。

=== コード例

このコード例は  _WinTree_   コントロールを含むプロジェクトが必要です。プロジェクトがない場合は印刷プレビューや印刷設定を実行する前に、 link:wintree.html[WinTree] ヘルプ ドキュメントを参照してください。

以下のコードは、ボタンのクリック イベントで印刷プレビューを実行する  _WinTree_   を示します。

*C# の場合:*

[source,csharp]
----
using (var printDoc = new UltraTreePrintDocument())
{
    printDoc.Tree = _ultraTree1;
    using (var previewDialog = new UltraPrintPreviewDialog())
    {
        previewDialog.Document = printDoc;
        previewDialog.ShowDialog(this);
    }
}
----

*Visual Basic の場合:*

[source,vb]
----
Using printDoc As UltraTreePrintDocument = New UltraTreePrintDocument()
printDoc.Tree = _ultraTree1
       Using previewDialog As UltraPrintPreviewDialog = New UltraPrintPreviewDialog()
             previewDialog.Document = printDoc
              previewDialog.ShowDialog(Me)
       End Using
End Using
----

印刷プレビュー ダイアログを開き、 _WinTree_   で印刷を実行する代わりに、 link:{ApiPlatform}win.ultrawintree{ApiVersion}~infragistics.win.ultrawintree.ultratreeprintdocument_members.html[UltraTreePrintDocument] の Print メソッドを呼び出します。

*C# の場合:*

[source,csharp]
----
using (UltraTreePrintDocument printDoc = new UltraTreePrintDocument())
{
    printDoc.Tree = ultraTree1;
    printDoc.Print();
}
----

*Visual Basic の場合:*

[source,vb]
----
Using printDoc As New UltraTreePrintDocument()
      printDoc.Tree = ultraTree1
      printDoc.Print()
End Using
----
