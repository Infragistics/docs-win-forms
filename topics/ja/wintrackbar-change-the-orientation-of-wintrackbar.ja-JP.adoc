= WinTrackBar の方向の変更

多くの Infragistics の Windows Forms コントロールおよびいくつかの Microsoft のインボックス コントロールは、水平および垂直の両方向にフォームで配置することができ、WinTrackBar は水平および垂直の両方向に向けることができるコントロールのこのグループに含まれます。pick:[win-forms="link:{ApiPlatform}win.ultrawineditors{ApiVersion}~infragistics.win.ultrawineditors.ultratrackbar~orientation.html[Orientation]"]  プロパティを設定することでこのアクションを簡単に実行できます。デフォルトでは、WinTrackBar は水平にフォームに置かれます。Orientation プロパティを Orientation.Vertical に設定すると、WinTrackBar が垂直方向に表示されます。

[cols="a,a"]
|====

|image::images/WinTrackBar_Change_the_Orientation_of_WinTrackBar_01.png[UltraTrackBar の Orientation プロパティを Horizontal に更新します] 

|image::images/WinTrackBar_Change_the_Orientation_of_WinTrackBar_02.png[UltraTrackBar の Orientation プロパティを Vertical に更新します] 

|Orientation.Horizontal (デフォルト)
|Orientation.Vertical

|====

WinTrackBar の方向を変更するには以下のコード例を使用します。

*Visual Basic の場合：*

----
' Set WinTrackBar's Orientation to Vertical.
 Me.UltraTrackBar1.Orientation = Orientation.Vertical
----

*C# の場合：*

----
// Set WinTrackBar's Orientation to Vertical.
 this.ultraTrackBar1.Orientation = Orientation.Vertical;
----
