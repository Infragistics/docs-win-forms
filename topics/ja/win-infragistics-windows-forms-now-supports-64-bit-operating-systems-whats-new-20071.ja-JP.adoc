= {ProductName} は 64 ビット オペレーティング システムをサポートする

これまでは、{ProductName} はすべての Microsoft® 32 ビット オペレーティング システムをサポートしていました。Windows® Vista™ の発売および 64 ビット オペレーションをサポートするデュアルおよびクアッド コア プロセッサの出現によって、広範なプロセッサおよびオペレーティング システムをサポートする能力をユーザーに提供することが急務となりました。

弊社は、Windows XP 64 ビット、Windows Vista 64 ビット、Windows Server 2003 64 ビット、および 64 ビット .NET 2.0 アーキテクチャで機能するすべてのオペレーティング システムを含むためにサポートされる範囲のオペレーティング システムを拡張しました。
