= Date の値が null の場合の WinCalendarCombo テキストの設定

pick:[win-forms="link:{ApiPlatform}win.ultrawinschedule{ApiVersion}~infragistics.win.ultrawinschedule.ultracalendarcombo~nulldatelabel.html[NullDateLabel]"]  プロパティを使用して、データ値が null のときに WinCalendarCombo テキストを設定します。

*Visual Basic の場合：*

----
Private Sub ChangeTextWhenDateValueIsNull_Load(ByVal sender As Object, _
  ByVal e As System.EventArgs) Handles MyBase.Load
	Me.UltraCalendarCombo1.NullDateLabel = "[Empty]"
End Sub
----

*C# の場合：*

----
private void ChangeTextWhenDateValueIsNull_Load(object sender, System.EventArgs e)
{
	this.ultraCalendarCombo1.NullDateLabel = "[Empty]";
}
