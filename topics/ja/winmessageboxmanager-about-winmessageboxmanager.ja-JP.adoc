= WinMessageBoxManager について

WinMessageBoxManager™ コンポーネントによってスタイルおよびカスタマイズ可能なメッセージ ボックスを通してエンドユーザーに情報を提供することができます。 メッセージ ボックスに表示される情報は、pick:[win-forms="link:{ApiPlatform}win{ApiVersion}~infragistics.win.ultramessagebox.ultramessageboxinfo.html[UltraMessageBoxInfo]"]  オブジェクトのインスタンスを通して定義できます。 WinMessageBoxManager コンポーネントの主要な機能は以下の通りです。

* アプリケーションに合うように、Appearance オブジェクトと AppStylist を使用してスタイルします。
* メッセージ ダイアログ ウィンドウ内のテキストは異なる色、フォントを適用する、画像を追加する、URL を設定するなどによってフォーマットできます。
* 個別のヘッダー、コンテンツ、フッターの各セクションはカスタマイズ可能な情報の表示のために個々にフォーマットできます。
