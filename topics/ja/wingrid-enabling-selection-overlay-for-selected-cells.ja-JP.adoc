= 選択セルの選択オーバーレイを有効にする

Microsoft® Excel® で、矩形の選択領域が選択されたセル上にオーバーレイされますが、このオーバーレイは元となるセルのカラーウォッシュした外観を提供します。pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridlayout.html[UltraGridLayout]"]  オブジェクトの pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridlayout~selectionoverlaycolor.html[SelectionOverlayColor]"]  プロパティと pick:[win-forms="link:{ApiPlatform}win.ultrawingrid{ApiVersion}~infragistics.win.ultrawingrid.ultragridlayout~selectionoverlaybordercolor.html[SelectionOverlayBorderColor]"]  プロパティを使用することによって、WinGrid™ で同じ効果を達成することができます。

[NOTE]
====
*注：* 選択オーバーレイの境界線が描画されるのは、単一の連続した選択の矩形領域を実行する時だけです。たとえば、それぞれ隣接しない 2 つの列セットを選択する場合、境界線は描画されません。同様に、さまざまなセル選択範囲や行選択範囲は、Excel® の動作方法と同じように境界線がなくなります。
====

*Visual Basic の場合：*

----
Me.ultraGrid1.DisplayLayout.SelectionOverlayColor = Color.DarkBlue
Me.ultraGrid1.DisplayLayout.SelectionOverlayBorderColor = Color.Magenta
----

*C# の場合：*

----
this.ultraGrid1.DisplayLayout.SelectionOverlayColor = Color.DarkBlue;
this.ultraGrid1.DisplayLayout.SelectionOverlayBorderColor = Color.Magenta;
----
