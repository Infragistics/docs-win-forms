= WinGridFilterUIProvider Excel スタイル フィルタリング

{ProductName} の 2011 Volume 2 以降、グリッドのフィルタリング機能は Microsoft Excel® のフィルタリング動作と同じに動作します。Excel では、セル値のリストがフィルターされ、他の列にフィルター アウトされていない値のみが表示されます。

フィルタリング機能は WinGridFilterUIProvider™ コンポーネントによって提供されます。以前のバージョンの WinGridFilterUIProvider は、フィルター アウトされた状態かどうかにかかわらず、ドロップダウン ツリーにすべての値を残しました。古い機能との後方互換性のために、UltraGridFilterUIProvider では link:{ApiPlatform}win.supportdialogs{ApiVersion}~infragistics.win.supportdialogs.filteruiprovider.ultragridfilteruiprovider~operandlistfilterbehavior.html[OperandListFilterBehavior] という名前の新しいプロパティが使用可能になりました。
