= Title Text プロパティのデフォルトは空の文字列

これまで、ユーザーからチャートのタイトルに関して混乱しているとの報告がありました（"Copyright Infragistics" や "Infragistics UltraChart" など）。これらのタイトルはチャートがライセンスされていないものであることを意味し、プロジェクト登録後これらのタイトルは消えるものだと憶測されることもありました。混乱を防ぐために、現在ではタイトルのテキスト プロパティはデフォルトで空白の文字列となっています。
