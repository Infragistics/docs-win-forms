= 通貨記号なしで通貨マスクを表示

pick:[win-forms="link:{ApiPlatform}win{ApiVersion}~infragistics.win.editorwithmask.html[EditorWithMask]"]  基本クラスおよび WinCurrencyEditor コントロールの両方で使用される既存の通貨マスクにフラグを追加しました。 _no_symbol_  フラグによって、実際の通貨記号（すなわち、¥, £, €, $）を使用することなく特定の通貨を指定することができます。たとえば、{currency:n.m:no_symbol} マスクを使用して、ドル記号を表示せずにアメリカの通貨のドルとセント（$5.50 は 5.50 と表示される）を表示することができます。これらのマスクの詳細は、pick:[win-forms="link:{ApiPlatform}win.ultrawineditors{ApiVersion}~infragistics.win.ultrawineditors.ultranumericeditor~maskinput.html[MaskInput]"]  プロパティを参照してください。
