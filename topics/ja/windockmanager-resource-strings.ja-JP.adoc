= WinDockManager リソース文字列

コントロール/コンポーネントによりユーザーに表示されるリソース文字列値を迅速にそして簡単に変更することができます。リソース文字列には、ダイアログ ボックスのキャプション、警告メッセージ、アラートなどがあります。

次の表は、文字列のデフォルト テキスト値と共に、このアセンブリによって使用されるリソース文字列名をリストします。

[options="header", cols="a,a"]

|リソース文字列名|デフォルト値

|AllowDropErrorMessage
|AllowDrop プロパティが true に設定されていますが、現在のアパートメント状態 (ApartmentState) は、ドラッグ アンド ドロップをサポートしていません。ドラッグ アンド ドロップ機能を無効にする場合は、AllowDrop プロパティを false に設定してください。ドラッグ アンド ドロップ機能を有効にする場合は、スレッドの ApartmentState プロパティを STAThread に設定してください。これは、Main メソッドに [STAThread] 属性を追加することでも実現できます。

|QuickHelpDlgBody3

|QuickHelpDlgBody4

|QuickHelpDlgBody5

|QuickHelpDlgBody6
|ドックされたコントロールがある場合、カスタム プロパティ ページに追加のプロパティ ウィンドウが表示されます。このプロパティ ウィンドウには、ドック コントロールに関連付けられた DockableControlPane のプロパティが表示されます。
