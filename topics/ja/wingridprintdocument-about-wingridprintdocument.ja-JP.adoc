= WinGridPrintDocument について

WinGridPrintDocument は WinGrid の印刷を可能にする Windows Form コンポーネントです。以下は WinGridPrintDocument で利用可能な機能の一部です。

* 印刷プレビュー -- WinGridPrintDocument コンポーネントは、WinGrid ドキュメントを印刷前にプレビューするために、WinPrintPreviewDialog コンポーネントとともに使用できます。詳細は、 link:winprintpreviewdialog-wingridprintdocument-using-winprintpreviewdialog-with-wingridprintdocument.html[「WinGridPrintDocument と WinPrintPreviewDialog を使用」]を参照してください。
* ヘッダ/フッタ -- WinGrid コントロールとともにカスタムのヘッダとフッタを印刷するように指定できます。
