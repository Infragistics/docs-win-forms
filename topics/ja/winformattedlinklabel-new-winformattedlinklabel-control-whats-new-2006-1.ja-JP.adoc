= 新しい WinFormattedLinkLabel コントロール

これまでは、WinGrid またはその他のコントロールで HTML 機能を適切に設定するには大掛かりな作業が必要でした。この作業を容易にするために新しいコントロール、WinFormattedLinkLabel が追加されました。このコントロールは、FormattedLinkEditor クラスを使用し、WinGrid、WinTree、およびその他の埋め込みエディタをサポートしているコントロールでのフォーマットされたテキストの表示を可能にします。

FormattedLinkEditor によるフォーマットされたテキストの表示を完全に制御できます。太字（<b>、<strong>）、斜体（<i>、<em>）、下線（<u>）、フォント（<font face>、<font color>、<font size>）など、テキストフォーマット用にいくつかの標準タグを使用できます（HTML タグと似ていますが、エディタでは妥当な XML タグとして表示されます）。また、リンクをクリックして LinkClicked イベントが発生すると、カスタム動作が実行されます。
