== UndoUnit クラス参照

[[_Ref320801583]]

=== はじめに

UndoUnit は、元に戻す/やり直し履歴に追加される元に戻すことができる操作の基本クラスです。

これは、操作の文字列説明を取得し、操作を実行し、オプションで操作の結果を結合するためのメソッドを公開します。

[[_derived]]
