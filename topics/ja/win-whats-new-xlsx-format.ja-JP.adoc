= XLSX フォーマット

Microsoft®Office 2007® で新しい Excel® ファイル フォーマットが採用されました。新しい xlsx ファイル タイプは業界で標準になっています。Workbook オブジェクトをファイル システムに保存するかどうかにかかわらず、または Excel Exporter を使用する場合には、新しい xlsx ファイル フォーマットを使用して Excel ファイルを作成およびロードすることが可能となりました。
