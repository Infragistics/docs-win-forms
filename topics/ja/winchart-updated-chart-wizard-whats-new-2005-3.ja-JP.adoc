= 更新されたグラフ ウィザード

合成チャートや標準的なチャートを作成するための新しいチャート機能を反映させるためにチャート ウィザードは更新されました。チャート ウィザードには標準的なチャートまたは新しい合成チャートすべてを完全に設定する支援を行う機能があり、一個所からすべてを使用できます。
